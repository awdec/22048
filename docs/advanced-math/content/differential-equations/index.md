<h1><center>第六章 常微分方程</center></h1>

## 1. 本章定位

常微分方程研究的是未知函数及其导数之间的关系。

普通函数问题通常是：

$$
y=f(x)
$$

微分方程问题通常是：

$$
F(x,y,y',y'',\cdots,y^{(n)})=0
$$

也就是先给出函数变化规律，再反求函数本身。

本章主线为：

$$
\text{微分方程基本概念}
\longrightarrow
\text{一阶微分方程}
\longrightarrow
\text{高阶微分方程}
\longrightarrow
\text{综合建模}
$$

::: tip 重点
数学二中，常微分方程重点在一阶常见类型、可降阶高阶方程、二阶常系数线性微分方程及简单应用。
:::

## 3. 学习建议

微分方程题目常见难点不在“求导”，而在识别方程类型。

| 题型 | 优先思路 |
| --- | --- |
| 能写成 $g(y)dy=f(x)dx$ | 分离变量 |
| 形如 $y'+P(x)y=Q(x)$ | 一阶线性微分方程 |
| 形如 $y'+P(x)y=Q(x)y^n$ | 伯努利方程 |
| 形如 $Mdx+Ndy=0$ 且 $M_y=N_x$ | 全微分方程 |
| 右端不显含 $y$ 或 $x$ | 考虑降阶 |
| 二阶常系数齐次 | 写特征方程 |
| 二阶常系数非齐次 | 先求齐次通解，再求特解 |
| 应用题 | 先设未知函数，再根据变化率列方程 |

::: warning 易错
微分方程求出的通解含任意常数；若题目给出初值条件，必须代入初值确定常数，得到特解。
:::
