<h1><center>第二章 一元函数微分学【重要度：★★★★★】</center></h1>

## 1. 本章定位

一元函数微分学研究函数在某一点附近的变化率，以及函数图形的局部形态。

第一章解决的是：

$$
\text{函数是否趋近某个值}
$$

第二章进一步研究：

$$
\text{函数变化得有多快、图形如何变化}
$$

本章是后续积分学、微分方程和应用题的基础。

::: tip 重点
导数是本章核心工具。单调性、极值、凹凸性、拐点、渐近线、曲率等内容，本质上都是利用导数研究函数图形。
:::

## 4. 常见题型

| 题型 | 常用方法 |
| --- | --- |
| 求导数 | 套求导法则，注意复合结构 |
| 分段函数可导性 | 分别算左右导数 |
| 隐函数求导 | 两边同时对 $x$ 求导 |
| 参数方程求导 | 用 $\frac{dy}{dx}=\frac{dy/dt}{dx/dt}$ |
| 证明不等式 | 构造函数，用单调性或中值定理 |
| 求极值最值 | 找驻点、不可导点和端点 |
| 判断凹凸性 | 看二阶导数符号 |
| 求渐近线 | 看无穷远处或无穷间断点 |
| 求曲率 | 套曲率公式 |

## 5. 易错提示

- 可导一定连续，连续不一定可导。
- 分段函数在分界点处要分别算左右导数。
- 隐函数求导时，$y$ 是 $x$ 的函数，求导会产生 $y'$。
- 参数方程求二阶导时，不能直接对 $t$ 再求一次。
- 极值点不一定是导数为 $0$ 的点，不可导点也可能是极值点。
- 拐点要求凹凸性改变，不只是 $f''(x)=0$。
- 渐近线要分铅直、水平、斜渐近线逐类判断。
