<h1><center>第三章 一元函数积分学</center></h1>

## 1. 本章定位

一元函数积分学研究“由变化率恢复函数”和“连续累积量”的计算。

微分学解决的是：

$$
\text{已知函数，研究变化率}
$$

积分学解决的是：

$$
\text{已知变化率或局部量，研究总量}
$$

本章主线为：

$$
\text{原函数}
\longrightarrow
\text{不定积分}
\longrightarrow
\text{定积分}
\longrightarrow
\text{变限积分}
\longrightarrow
\text{积分应用}
$$

::: tip 重点
积分学的核心是两件事：会算积分，会把实际问题转化为积分。
:::

## 3. 学习建议

积分学题目常见难点不在公式本身，而在方法选择。

| 题型 | 优先思路 |
| --- | --- |
| 不定积分 | 识别结构，选择换元或分部 |
| 定积分 | 先看对称性、周期性，再计算 |
| 变限积分 | 先求导，再处理上下限 |
| 反常积分 | 先写成极限，再判断敛散 |
| 几何应用 | 先画图，确定积分变量和积分区间 |

::: warning 易错
不定积分结果要加常数 $C$，定积分结果是数值，不加 $C$。
:::
