<h1><center>高等数学总复习总结</center></h1>

## 1. 高等数学的总主线

考研数学二高等数学部分，可以理解为围绕“函数”展开的六条主线。

$$
\text{函数}
\longrightarrow
\text{极限}
\longrightarrow
\text{导数}
\longrightarrow
\text{积分}
\longrightarrow
\text{多元函数}
\longrightarrow
\text{微分方程}
$$

其中：

- 极限是基础语言。
- 导数研究变化率。
- 积分研究累积量。
- 多元函数研究多个变量共同变化。
- 微分方程研究由变化规律反求函数。

::: tip 总体判断
高数复习不是六章孤立背公式，而是从“函数如何变化、如何累积、如何反求”这三件事往外展开。
:::

## 2. 六章内容总览

| 章节 | 核心问题 | 常考能力 |
| --- | --- | --- |
| 第一章 函数 极限 连续 | 函数是否趋近某个值 | 极限计算、连续性、间断点 |
| 第二章 一元函数微分学 | 函数变化得多快 | 求导、单调性、极值、中值定理 |
| 第三章 一元函数积分学 | 局部量如何累积为整体量 | 积分计算、变限积分、反常积分、几何应用 |
| 第四章 多元函数微分学 | 多变量函数如何变化 | 偏导、全微分、隐函数、极值最值 |
| 第五章 二重积分 | 平面区域上的累积量 | 写积分区域、换次序、极坐标、应用 |
| 第六章 常微分方程 | 已知变化规律如何求函数 | 方程分类、通解、特解、建模 |

## 3. 第一章：函数、极限与连续

第一章是整个高数的地基。

核心链条：

$$
\text{函数性质}
\longrightarrow
\text{极限}
\longrightarrow
\text{无穷小}
\longrightarrow
\text{连续}
\longrightarrow
\text{间断}
$$

### 3.1 必会内容

| 内容 | 重点 |
| --- | --- |
| 函数性质 | 奇偶性、单调性、周期性、有界性 |
| 极限定义 | 左右极限、数列极限、函数极限 |
| 无穷小 | 等价无穷小、同阶、高阶、低阶 |
| 极限计算 | 等价替换、洛必达、泰勒、夹逼、重要极限 |
| 连续性 | 极限值等于函数值 |
| 间断点 | 可去、跳跃、无穷、振荡 |

### 3.2 第一章解题入口

看到极限题，先判断类型：

1. 能否直接代入。
2. 是否为未定式。
3. 是否能等价无穷小替换。
4. 是否适合洛必达法则。
5. 是否需要泰勒展开。
6. 是否用夹逼准则或单调有界准则。

::: warning 易错
等价无穷小替换通常用于乘除结构，不能随意在加减结构中局部替换。
:::

## 4. 第二章：一元函数微分学

第二章研究函数的变化率和图形性质。

核心链条：

$$
\text{导数定义}
\longrightarrow
\text{导数计算}
\longrightarrow
\text{中值定理}
\longrightarrow
\text{函数性态}
\longrightarrow
\text{最值应用}
$$

### 4.1 必会内容

| 内容 | 重点 |
| --- | --- |
| 导数定义 | 差商极限、左右导数 |
| 可导与连续 | 可导必连续，连续不一定可导 |
| 求导法则 | 复合、隐函数、参数方程、高阶导数 |
| 中值定理 | 罗尔、拉格朗日、柯西 |
| 洛必达法则 | 处理未定式极限 |
| 单调性 | 看 $f'(x)$ 符号 |
| 极值最值 | 驻点、不可导点、端点 |
| 凹凸性 | 看 $f''(x)$ 符号 |
| 渐近线 | 水平、垂直、斜渐近线 |
| 曲率 | 曲率与曲率半径 |

### 4.2 第二章解题入口

| 题型 | 入口 |
| --- | --- |
| 求切线法线 | 求导数，得到斜率 |
| 判断单调区间 | 解 $f'(x)>0$ 或 $f'(x)<0$ |
| 求极值 | 找 $f'=0$ 和不可导点 |
| 求最值 | 比较内部候选点和端点 |
| 判断凹凸 | 看 $f''(x)$ 正负 |
| 中值定理证明 | 构造函数，验证连续可导 |
| 含参数讨论 | 分界点、导数符号、连续可导条件 |

::: tip 重点
导数计算是工具，中值定理和函数性态才是第二章的大题核心。
:::

## 5. 第三章：一元函数积分学

第三章研究累积量和反求原函数。

核心链条：

$$
\text{原函数}
\longrightarrow
\text{不定积分}
\longrightarrow
\text{定积分}
\longrightarrow
\text{变限积分}
\longrightarrow
\text{反常积分}
\longrightarrow
\text{积分应用}
$$

### 5.1 必会内容

| 内容 | 重点 |
| --- | --- |
| 不定积分 | 基本公式、换元、分部 |
| 定积分 | 牛顿-莱布尼茨公式 |
| 变限积分 | 上下限求导公式 |
| 定积分性质 | 对称性、周期性、比较、估值 |
| 反常积分 | 无穷区间、无界函数、敛散性 |
| 几何应用 | 面积、体积、弧长 |

### 5.2 第三章方法选择

| 积分特征 | 方法 |
| --- | --- |
| 内层函数与导数同时出现 | 凑微分 |
| 多项式乘指数或三角函数 | 分部积分 |
| 对称区间 | 先看奇偶性 |
| 周期函数 | 利用周期性质 |
| 上限含变量 | 用变限积分求导 |
| 积分端点异常 | 写成反常积分极限 |
| 几何面积体积 | 先画图，再取微元 |

::: warning 易错
不定积分结果要加 $C$；定积分结果是数值，不加 $C$。
:::

## 6. 第四章：多元函数微分学

第四章把“变化率”从一元推广到多元。

核心链条：

$$
\text{二元函数}
\longrightarrow
\text{二元极限与连续}
\longrightarrow
\text{偏导数}
\longrightarrow
\text{全微分}
\longrightarrow
\text{复合与隐函数求导}
\longrightarrow
\text{极值最值}
$$

### 6.1 必会内容

| 内容 | 重点 |
| --- | --- |
| 二元极限 | 多路径趋近 |
| 连续性 | 极限值等于函数值 |
| 偏导数 | 固定另一个变量 |
| 全微分 | $dz=f_xdx+f_ydy$ |
| 可微关系 | 可微推出连续和偏导存在 |
| 复合函数求导 | 沿路径相乘，不同路径相加 |
| 隐函数求导 | 公式法或两边求导 |
| 极值 | 驻点与二阶判别式 |
| 条件极值 | 拉格朗日乘数法 |
| 闭区域最值 | 内部、边界、角点全部比较 |

切平面与法线可作为全微分几何意义的拓展理解，不必放在数学二主线复习的最前面。

### 6.2 多元函数逻辑关系

需要牢牢记住：

$$
\text{可微}
\Rightarrow
\text{连续}
$$

$$
\text{可微}
\Rightarrow
f_x,\ f_y\text{ 存在}
$$

但：

$$
f_x,\ f_y\text{ 存在}
\nRightarrow
\text{连续}
$$

$$
f_x,\ f_y\text{ 存在}
\nRightarrow
\text{可微}
$$

::: warning 易错
偏导存在只说明沿坐标轴方向变化率存在，不代表所有方向都有统一的线性近似。
:::

## 7. 第五章：二重积分

第五章把“累积”从区间推广到平面区域。

核心链条：

$$
\text{积分区域}
\longrightarrow
\text{面积微元}
\longrightarrow
\text{累次积分}
\longrightarrow
\text{换次序}
\longrightarrow
\text{极坐标}
\longrightarrow
\text{应用}
$$

### 7.1 必会内容

| 内容 | 重点 |
| --- | --- |
| 二重积分定义 | 平面区域上的和式极限 |
| 几何意义 | 曲顶柱体体积 |
| 性质 | 线性、区域可加、比较、估值、对称 |
| 直角坐标 | $X$ 型区域、$Y$ 型区域 |
| 交换次序 | 先画区域，再改写积分限 |
| 极坐标 | $d\sigma=rdrd\theta$ |
| 应用 | 面积、体积、质量、质心、转动惯量 |

### 7.2 二重积分解题顺序

1. 画出区域 $D$。
2. 判断是否有对称性。
3. 判断用直角坐标还是极坐标。
4. 写出积分限。
5. 检查面积微元。
6. 计算累次积分。

::: tip 重点
二重积分的第一步永远是区域。区域决定坐标系、积分限和是否需要拆分。
:::

## 8. 第六章：常微分方程

第六章研究“由变化规律反求函数”。

核心链条：

$$
\text{方程类型}
\longrightarrow
\text{通解}
\longrightarrow
\text{初值条件}
\longrightarrow
\text{特解}
\longrightarrow
\text{应用建模}
$$

### 8.1 必会内容

| 类型 | 方法 |
| --- | --- |
| 可分离变量 | 分离变量后积分 |
| 一阶线性 | 积分因子 |
| 伯努利方程 | 令 $z=y^{1-n}$ |
| 齐次方程 | 令 $y=ux$ |
| 全微分方程 | 判断 $M_y=N_x$，求 $u(x,y)=C$ |
| 可降阶高阶方程 | 令 $p=y'$ |
| 二阶常系数齐次 | 特征方程 |
| 二阶常系数非齐次 | 齐次通解加一个特解 |
| 高于二阶常系数齐次 | 高阶特征方程 |
| 综合题 | 先建模，再求解 |

### 8.2 常见建模入口

| 题目语言 | 方程语言 |
| --- | --- |
| 变化率与自身成正比 | $\frac{dy}{dt}=ky$ |
| 变化率与剩余量成正比 | $\frac{dy}{dt}=k(A-y)$ |
| 流入流出 | $\frac{dy}{dt}=\text{流入}-\text{流出}$ |
| 切线斜率 | $y'=F(x,y)$ |
| 积分方程 | 先对 $x$ 求导 |

::: warning 易错
微分方程有初值条件时，最后必须求出特解；不要停在含任意常数的通解。
:::

## 9. 全局解题策略

### 9.1 计算题

计算题的核心是识别结构。

| 看到什么 | 想到什么 |
| --- | --- |
| $0/0,\ \infty/\infty$ | 洛必达或泰勒 |
| 小量相乘相除 | 等价无穷小 |
| 函数单调极值 | 求导 |
| 面积、体积、累积 | 积分 |
| 上限含变量的积分 | 变限积分求导 |
| 区域积分 | 二重积分 |
| 圆域、扇形 | 极坐标 |
| 变化率关系 | 微分方程 |

### 9.2 证明题

证明题的核心是构造。

| 题目特征 | 常用工具 |
| --- | --- |
| 证明存在零点 | 介值定理 |
| 证明存在导数为零的点 | 罗尔定理 |
| 出现 $f(b)-f(a)$ | 拉格朗日中值定理 |
| 不等式证明 | 构造函数 + 单调性 |
| 积分不等式 | 比较性质、估值、中值定理 |
| 极值存在 | 连续函数在有界闭区域上的最值定理 |

### 9.3 应用题

应用题的核心是翻译。

| 现实语言 | 数学语言 |
| --- | --- |
| 瞬时变化率 | 导数 |
| 总量、累积量 | 积分 |
| 平面区域累积 | 二重积分 |
| 最优问题 | 极值或最值 |
| 约束条件下最优 | 拉格朗日乘数法 |
| 按规律变化 | 微分方程 |

## 10. 全局易错点

1. 极限题直接套公式，不先判断未定式。
2. 等价无穷小在加减结构中乱替换。
3. 分段函数不检查分界点。
4. 可导、连续、可微之间逻辑关系混淆。
5. 中值定理忘记验证闭区间连续、开区间可导。
6. 不定积分忘记加 $C$。
7. 定积分换元忘记改上下限。
8. 反常积分不先写成极限。
9. 二重积分不画区域，导致积分限写错。
10. 极坐标漏乘 $r$。
11. 多元函数最值题漏掉边界。
12. 微分方程有初值条件却没求特解。

## 11. 复习路线建议

### 第一轮：打通概念

目标是知道每章在解决什么问题。

重点：

- 函数、极限、连续的基本概念。
- 导数和积分的定义。
- 偏导、全微分、二重积分的几何意义。
- 微分方程的通解与特解。

### 第二轮：训练方法

目标是看到题能迅速分类。

重点：

- 极限方法分类。
- 导数应用分类。
- 积分方法分类。
- 二重积分区域分类。
- 微分方程类型分类。

### 第三轮：综合与查漏

目标是提高稳定性和速度。

重点：

- 中值定理证明。
- 含参数讨论。
- 积分综合题。
- 多元函数极值最值。
- 二重积分换次序与极坐标。
- 微分方程建模。

::: tip 最后提醒
高数不是靠“多背几个公式”堆起来的，而是靠题型识别和方法选择串起来的。每次做题都问一句：这道题到底是在考极限、变化率、累积量，还是反求函数？
:::
