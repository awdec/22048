<h1><center>线性表</center></h1>

### 1. 

顺序表可以利用一维数组表示，因此顺序表与一维数组在逻辑结构上是相同的。

**错**

逻辑结构：
- 集合结构
- 线性结构
- 树形结构
- 图状结构

都可以用数组一维数组存，所以顺序表不一定一样。

### 2.

顺序表交换两个相邻元素效率高于链表。

虽然都是 $O(1)$，但是操作次数不同。

### 3. 

线性表在创建时申请空间。

空间不足时，需要重新一片完整的连续空间。

### 4.

$T=O(n),M=O(1)$ 实现线性表删除所有值为 $x$。

从前向后扫，维护一个位置指针。如果 $a_i$ 不是 $x$，则放到当前指的位置上，指针右移。

### 5.

把 $a_1,\cdots,a_n,b_1,\cdots,b_m$ 修改为 $b_1,\cdots,b_m,a_1,\cdots,a_n$。

先翻转成 $b_m,\cdots,b_1,a_n,\cdots,a_1$。

再翻转 $b_1,\cdots,b_m,a_n,\cdots,a_1$。

最后翻转为 $b_1,\cdots,b_m,a_1,\cdots,a_n$。

### 6.

给定三个有序序列，找到 $|a_i-b_j|+|b_j-c_k|+|c_k-a_i|$ 的最大值。

首先注意到，求和式与 $a,b,c$ 三个值的顺序无关

答案就是 $2\times(\max\{a_i,b_j,c_k\}-\min\{a_i,b_j,c_k\})$

所以从 $a_1,b_1,c_1$，每次把最小的那一个数的指针右移。

$T=O(n),M=O(1)$。
