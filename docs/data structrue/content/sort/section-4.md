<h1><center>第四节 选择排序</center></h1>

## 1. 选择排序

选择排序的基本思想是：每趟从无序区中选择一个最小或最大记录，放到最终位置。

408 中常见选择排序：

- 简单选择排序
- 堆排序

## 2. 简单选择排序

简单选择排序每趟从无序区中选出最小元素，与无序区第一个元素交换。

### 2.1 算法过程

对序列：

```txt
49 38 65 97 76 13
```

第一趟选择最小元素 $13$，与第一个元素交换：

```txt
13 38 65 97 76 49
```

此时第一个位置已经确定。

之后继续在剩余无序区中选择最小元素。

### 2.2 代码实现

::: details 点击展开代码

```c
void SelectSort(int A[], int n) {
    for (int i = 0; i < n - 1; i++) {
        int min = i;

        for (int j = i + 1; j < n; j++) {
            if (A[j] < A[min]) {
                min = j;
            }
        }

        if (min != i) {
            int temp = A[i];
            A[i] = A[min];
            A[min] = temp;
        }
    }
}
```

:::

### 2.3 性能分析

简单选择排序比较次数与初始序列无关。

时间复杂度为：

$$
O(n^2)
$$

空间复杂度为：

$$
O(1)
$$

稳定性：不稳定。

## 3. 堆排序

堆是满足特定性质的完全二叉树。

大根堆满足：

$$
parent \ge child
$$

小根堆满足：

$$
parent \le child
$$

若要得到升序序列，通常建立大根堆。

### 3.1 建堆

堆使用顺序存储。

若数组下标从 $1$ 开始，则结点 $i$ 的孩子为：

$$
2i,\quad 2i + 1
$$

建堆时，从最后一个分支结点开始向前调整。

最后一个分支结点下标为：

$$
\lfloor n / 2 \rfloor
$$

### 3.2 堆排序过程

升序堆排序过程：

1. 建立大根堆。
2. 将堆顶元素与堆尾元素交换。
3. 堆的有效长度减一。
4. 对新的堆顶向下调整。
5. 重复直到堆中只剩一个元素。

### 3.3 向下调整

::: details 点击展开代码

```c
void HeadAdjust(int A[], int k, int len) {
    A[0] = A[k];

    for (int i = 2 * k; i <= len; i *= 2) {
        if (i < len && A[i] < A[i + 1]) {
            i++;
        }

        if (A[0] >= A[i]) {
            break;
        } else {
            A[k] = A[i];
            k = i;
        }
    }

    A[k] = A[0];
}
```

:::

这段代码假设有效元素存放在 `A[1]` 到 `A[len]` 中，`A[0]` 作为暂存单元。

### 3.4 性能分析

建堆时间复杂度为：

$$
O(n)
$$

堆排序总时间复杂度为：

$$
O(n\log n)
$$

空间复杂度为：

$$
O(1)
$$

稳定性：不稳定。

## 4. 简单选择排序与堆排序比较

| 项目 | 简单选择排序 | 堆排序 |
| --- | --- | --- |
| 思想 | 每趟选最小 | 每趟输出堆顶 |
| 时间复杂度 | $O(n^2)$ | $O(n\log n)$ |
| 空间复杂度 | $O(1)$ | $O(1)$ |
| 稳定性 | 不稳定 | 不稳定 |

## 5. 408 常考点

### 5.1 简单选择排序不稳定

选择并交换最小元素时，可能跨过相等关键字，导致相对次序改变。

### 5.2 堆是完全二叉树

堆不是普通二叉树，而是完全二叉树的顺序存储形式。

### 5.3 大根堆和升序

用大根堆排序，每次把最大元素放到当前无序区末尾，最终得到升序序列。

### 5.4 建堆从最后一个分支结点开始

若下标从 $1$ 开始，最后一个分支结点是 $\lfloor n/2 \rfloor$。

## 6. 小结

| 算法 | 时间复杂度 | 空间复杂度 | 稳定性 |
| --- | --- | --- | --- |
| 简单选择排序 | $O(n^2)$ | $O(1)$ | 不稳定 |
| 堆排序 | $O(n\log n)$ | $O(1)$ | 不稳定 |

选择排序的核心是：每趟选择一个元素放到最终位置。
