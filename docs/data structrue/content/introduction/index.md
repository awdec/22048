<h1><center>第一章 绪论</center></h1>

## 1. 本章定位

绪论是数据结构的入口章节，主要回答两个问题：

1. 数据结构研究什么。
2. 算法应该如何评价。

408 数据结构后续所有章节，都会围绕下面这条主线展开：

$$
\text{逻辑结构}
\longrightarrow
\text{存储结构}
\longrightarrow
\text{基本运算}
\longrightarrow
\text{算法分析}
$$

::: tip 重点
数据结构关注“数据之间的关系及其存储表示”，算法关注“解决问题的步骤及其效率”。
:::

## 2. 本章小节

### 第一节 基本概念

本节主要整理数据结构的基本术语和结构分类：

- 数据
- 数据元素
- 数据项
- 数据对象
- 数据结构
- 逻辑结构
- 存储结构
- 数据运算

重点是理解数据结构的三要素：

$$
\text{逻辑结构}
\quad
\text{存储结构}
\quad
\text{数据的运算}
$$

### 第二节 算法概念

本节主要整理算法的基本概念和复杂度分析：

- 算法的定义
- 算法的五个特性
- 好算法的标准
- 时间复杂度
- 空间复杂度

重点是会根据代码或伪代码估计算法复杂度。

## 3. 学习建议

绪论内容概念较多，复习时不要只背名词，应重点建立对应关系：

| 问题 | 对应内容 |
| --- | --- |
| 数据之间是什么关系 | 逻辑结构 |
| 数据在内存中怎么放 | 存储结构 |
| 对数据做什么操作 | 数据运算 |
| 算法运行快不快 | 时间复杂度 |
| 算法占用空间多不多 | 空间复杂度 |

后续学习线性表、栈、队列、树、图时，都可以按这个框架复习。
