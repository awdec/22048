<h1><center>第二节 分页、分段与地址转换</center></h1>

::: info 例 1
页面大小为 $1\text{ KB}$，逻辑地址为 $2500$，求页号和页内偏移。
:::

::: details 解析
$1\text{ KB}=1024$ 字节：
$$
p=\left\lfloor\frac{2500}{1024}\right\rfloor=2,
$$
$$
w=2500-2\times1024=452.
$$
:::

::: info 例 2
上题中页 $2$ 位于页框 $7$，求物理地址。
:::

::: details 解析
$$
PA=7\times1024+452=7620.
$$
:::

::: info 例 3
TLB 查询时间为 $10\text{ ns}$，内存访问时间为 $100\text{ ns}$，命中率为 $90\%$。忽略缺页，求有效访问时间。
:::

::: details 解析
命中需 $10+100=110\text{ ns}$，未命中需 $10+200=210\text{ ns}$：
$$
EAT=0.9\times110+0.1\times210=120\text{ ns}.
$$
:::

::: info 例 4
分页和分段中，哪一种更方便按逻辑模块共享和保护？
:::

::: details 解析
分段。段由程序的逻辑模块构成，段表项可按模块设置长度与权限；页是系统等长划分的物理管理单位，对用户通常不可见。
:::
