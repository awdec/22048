<h1><center>第一节 内存基础与连续分配</center></h1>

::: info 例 1
某进程的基址寄存器为 $12000$，界限寄存器为 $4000$。逻辑地址 $3500$ 和 $4500$ 的访问结果分别是什么？
:::

::: details 解析
$3500<4000$，地址合法，物理地址为：
$$
12000+3500=15500.
$$
$4500\ge4000$，超出进程地址范围，触发越界异常。
:::

::: info 例 2
动态分区产生许多分散的小空闲区，属于什么碎片？如何缓解？
:::

::: details 解析
属于外部碎片。可通过紧凑移动已分配分区，把空闲空间合并为连续区域，但需要动态重定位支持并产生移动开销。
:::

::: info 例 3
最佳适应算法为什么不一定“最佳”？
:::

::: details 解析
它每次选择最小可满足空闲区，容易留下大量难以利用的小碎片；“最佳”只描述当次剩余空间最小，不保证长期整体利用率最佳。
:::
