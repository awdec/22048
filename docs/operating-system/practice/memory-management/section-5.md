::: info 综合题
某分页系统逻辑地址为 $16$ 位，页面大小为 $1\text{ KB}$，页表项为 $4$ 字节。求逻辑页数和单级页表大小。
:::

::: details 解析
逻辑地址空间大小为 $2^{16}=64\text{ KB}$，每页 $2^{10}$ 字节，因此页数：
$$
2^{16-10}=64.
$$
页表大小：
$$
64\times4=256\text{ B}.
$$
:::

::: info 综合题 2
页面大小为 $4\text{ KB}$，一个进程大小为 $18\text{ KB}$，采用纯分页时最多产生多少内部碎片？本进程实际产生多少？
:::

::: details 解析
任一进程最后一页最多浪费不足一页，即小于 $4\text{ KB}$。该进程需要 $5$ 页，共 $20\text{ KB}$，实际内部碎片为：
$$
20-18=2\text{ KB}.
$$
:::

## 答题技巧

- 地址题先统一字节单位，再拆页号和偏移。
- TLB 未命中不等于缺页。
- 置换题画固定页框表，每次明确命中、装入和淘汰对象。
- 多级页表题分清各级索引位数和一次地址转换的访存次数。
