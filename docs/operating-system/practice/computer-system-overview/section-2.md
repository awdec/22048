<h1><center>第二节 运行环境与中断</center></h1>

::: info 例 1
缺页、时钟到时和系统调用分别属于哪类中断或异常？
:::

::: details 解析
- 缺页：由当前指令访存触发的故障，属于异常。
- 时钟到时：来自处理器外部的外中断。
- 系统调用：程序主动执行陷入指令，属于陷阱。
:::

::: info 例 2
应用程序能否直接执行关闭中断和设备 I/O 指令？
:::

::: details 解析
不能。这些是特权指令，只能在内核态执行。用户程序必须通过系统调用请求内核完成相关服务。
:::

::: info 例 3
发生外部中断后，处理器是否一定立即切换到另一个用户进程？
:::

::: details 解析
不一定。处理器先进入内核态执行中断处理程序，随后调度器根据事件和调度策略决定恢复原进程还是切换到其他进程。
:::
