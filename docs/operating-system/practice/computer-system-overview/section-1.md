<h1><center>第一节 概念、功能与特征</center></h1>

::: info 例 1
单核计算机的两个进程能否并发？能否并行？
:::

::: details 解析
可以并发：两个进程在一段时间内交替推进。不能在同一时刻真正并行执行两条用户指令，因为单核同一时刻只能运行一个执行流。
:::

::: info 例 2
多个进程可以同时读取同一只读文件，这属于互斥共享还是同时共享？
:::

::: details 解析
属于同时共享。多个进程的读取操作在宏观上一段时间内共同发生，不要求对整个文件实行排他访问。
:::

::: info 例 3
时间片轮转主要体现操作系统的哪两个特征？
:::

::: details 解析
主要体现并发和虚拟。多个进程交替执行形成并发，每个进程都产生仿佛独占处理器的虚拟效果。
:::
