<h1><center>第二节 处理机调度</center></h1>

::: info 例 1
三个进程同时到达，服务时间分别为 $6,2,4$。按非抢占 SJF 调度，求平均周转时间。
:::

::: details 解析
执行顺序为 $2\rightarrow4\rightarrow6$。完成时间分别为 $2,6,12$，到达时间均为 $0$，故平均周转时间：
$$
\frac{2+6+12}{3}=\frac{20}{3}.
$$
:::

::: info 例 2
三个进程 $P_1,P_2,P_3$ 同时到达，服务时间分别为 $5,3,1$，时间片为 $2$，画出 RR 的执行序列。
:::

::: details 解析
按到达队列顺序：
$$
P_1(2)\rightarrow P_2(2)\rightarrow P_3(1)\rightarrow
P_1(2)\rightarrow P_2(1)\rightarrow P_1(1).
$$
对应时间区间为 $0$–$2$、$2$–$4$、$4$–$5$、$5$–$7$、$7$–$8$、$8$–$9$。
:::

::: info 例 3
某作业等待 $6$，服务时间为 $2$，其响应比是多少？
:::

::: details 解析
$$
R_p=\frac{6+2}{2}=4.
$$
:::

::: info 例 4
多处理机调度为什么要同时考虑负载均衡和处理器亲和性？
:::

::: details 解析
负载均衡避免某些处理器繁忙而另一些空闲；处理器亲和性让任务尽量继续使用原处理器，以利用已有高速缓存。频繁迁移虽然可能平衡负载，却会破坏缓存局部性。
:::

::: info 例 5
系统没有普通就绪任务时，处理器是否可以执行任意阻塞进程？
:::

::: details 解析
不能。阻塞进程尚未具备运行条件。此时调度器选择闲逛进程，使处理器保持在受内核控制的可运行状态，直到中断等事件使普通任务就绪。
:::
