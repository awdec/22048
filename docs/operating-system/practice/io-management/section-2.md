<h1><center>第二节 缓冲、分配与 SPOOLing</center></h1>

::: info 例 1
设备输入一块数据耗时 $T$，进程处理一块耗时 $C$。忽略首次填充和切换开销，双缓冲稳定阶段处理每块数据约需多久？
:::

::: details 解析
输入与处理可并行，稳定阶段吞吐量由较慢者决定，因此每块约需：
$$
\max(T,C).
$$
:::

::: info 例 2
磁盘高速缓存和磁盘缓冲区的主要目的相同吗？
:::

::: details 解析
不完全相同。缓存保留数据副本，利用局部性避免再次访问慢速磁盘；缓冲主要协调传输速度、批量和数据粒度，数据通常要继续交付给接收方。
:::

::: info 例 3
SPOOLing 能否把打印机变成真正可并行工作的多台物理打印机？
:::

::: details 解析
不能。它把多个作业先写入磁盘输出井，再由守护进程顺序送往一台打印机，只是在逻辑上让多个进程共享独占设备。
:::
