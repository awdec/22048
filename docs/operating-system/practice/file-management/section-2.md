<h1><center>第二节 逻辑与物理结构</center></h1>

::: info 例 1
哪种文件分配方式最适合高效随机访问？连续分配的主要代价是什么？
:::

::: details 解析
连续分配可由起始块号直接计算任意逻辑块位置，随机访问最快；代价是产生外部碎片、文件增长困难，并可能需要预先知道大小。
:::

::: info 例 2
磁盘块大小为 $1\text{ KB}$，块号占 $4$ 字节，一个一级间接索引块可指向多少数据块？
:::

::: details 解析
$$
\frac{1024}{4}=256
$$
个数据块，对应数据容量 $256\text{ KB}$。
:::

::: info 例 3
某 inode 有 $10$ 个直接地址和一个一级间接地址，块大小 $4\text{ KB}$、块号 $4$ 字节，最大文件大小是多少？
:::

::: details 解析
一个间接块可保存 $4096/4=1024$ 个块号，总数据块数：
$$
10+1024=1034.
$$
最大文件大小：
$$
1034\times4\text{ KB}=4136\text{ KB}.
$$
:::
