<h1><center>第四章综合训练</center></h1>

::: info 综合题
某文件采用两级索引分配，块大小为 $1\text{ KB}$，每个块号占 $4$ 字节，不考虑直接地址。最大文件大小是多少？
:::

::: details 解析
每个索引块可保存：
$$
K=1024/4=256
$$
个块号。两级索引可定位 $256^2$ 个数据块，因此最大大小：
$$
256^2\times1\text{ KB}=64\text{ MB}.
$$
:::

::: info 综合题 2
访问两级索引文件的某数据块，若 inode 已在内存、两个索引块和数据块均未缓存，至少需要几次磁盘访问？
:::

::: details 解析
需要依次读取一级索引块、二级索引块和目标数据块，共 $3$ 次磁盘访问。
:::

## 答题技巧

- 索引题先求“一个索引块能放多少块号”。
- 最大文件大小只乘数据块容量，不把索引块算作文件数据。
- 路径题区分目录项、inode、系统打开文件表和进程描述符表。
