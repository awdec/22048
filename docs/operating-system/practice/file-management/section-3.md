<h1><center>第三节 空间管理与保护</center></h1>

::: info 例 1
磁盘共有 $2^{20}$ 个块，位示图至少需要多少空间？
:::

::: details 解析
每块占一位：
$$
2^{20}\text{ bit}=2^{17}\text{ B}=128\text{ KB}.
$$
:::

::: info 例 2
硬链接和符号链接的目标文件被删除后分别会怎样？
:::

::: details 解析
硬链接与原目录项平等地引用同一 inode，删除一个名字后其他硬链接仍有效；符号链接保存路径，目标路径被删除后会成为悬空链接。
:::

::: info 例 3
“某文件允许用户 A 读、用户 B 读写”更适合用 ACL 还是能力表表示？
:::

::: details 解析
更适合 ACL，因为它以文件对象为中心列出各主体权限。能力表则以用户或进程为中心列出其可访问的对象。
:::

::: info 例 4
为什么同一个 `open` 接口可以访问不同类型的文件系统？
:::

::: details 解析
VFS 向系统调用层提供统一抽象，再把操作转发给具体文件系统的实现。不同文件系统只需按 VFS 约定实现相应操作，应用程序不必了解其磁盘布局。
:::

::: info 例 5
把一个文件系统挂载到非空目录后，访问该目录会看到什么？
:::

::: details 解析
该目录成为挂载点，路径解析会进入被挂载文件系统；原目录中的内容暂时被遮蔽，但并未被删除，卸载后可再次访问。
:::
