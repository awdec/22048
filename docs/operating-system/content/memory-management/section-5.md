<h1><center>第三章总结：内存管理</center></h1>

## 地址转换公式

页面大小为 $L$：

$$
p=\left\lfloor\frac{A}{L}\right\rfloor,
\qquad w=A\bmod L,
$$

$$
PA=f\times L+w.
$$

## 碎片速查

| 分配方式 | 主要碎片 |
| --- | --- |
| 固定分区 | 内部碎片 |
| 动态分区 | 外部碎片 |
| 分页 | 页内内部碎片 |
| 分段 | 外部碎片 |

## 置换算法速查

| 算法 | 淘汰对象 | Belady 异常 |
| --- | --- | --- |
| OPT | 未来最晚使用 | 无 |
| FIFO | 最早进入 | 可能有 |
| LRU | 最久未访问 | 无 |
| CLOCK | 访问位为 0 | LRU 近似 |

- 页框不足时系统要分配或回收页框，修改页回收前通常需要写回。
- 内存映射文件把文件区域关联到进程虚拟地址空间，按需缺页调入。

::: tip 一句话总结
内存管理先完成地址转换与保护，再用虚拟内存扩大逻辑空间；缺页与置换是以较慢外存换取较大地址空间的代价。
:::
