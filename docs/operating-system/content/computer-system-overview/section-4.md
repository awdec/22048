<h1><center>第一章总结：计算机系统概述</center></h1>

## 核心对比

| 概念 | 关键点 |
| --- | --- |
| 并发 | 同一时间间隔内交替推进 |
| 并行 | 同一时刻同时执行 |
| 用户态 | 非特权指令、受保护运行 |
| 内核态 | 可执行特权指令 |
| 外中断 | 来自当前指令之外，异步 |
| 异常 | 由当前指令触发，同步 |
| 系统调用 | 用户程序请求内核服务的程序接口 |
| 外核 | 内核保护和分配资源，管理策略更多放到用户态 |
| 虚拟机 | 在同一物理机上提供相互隔离的完整执行环境 |

## 高频判断

1. 时钟中断可让操作系统重新获得处理器控制权。
2. 缺页属于异常，不是外中断。
3. 系统调用会引起用户态到内核态切换，但函数调用不一定。
4. 中断处理程序运行在内核态。
5. 微内核并非所有操作系统服务都在内核态运行。
6. 引导程序负责装入内核，内核随后初始化中断、内存和设备管理。

::: tip 一句话总结
操作系统通过中断接管处理器、通过系统调用提供服务，并以并发和共享为核心组织全部资源管理活动。
:::
