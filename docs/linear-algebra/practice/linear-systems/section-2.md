<h1><center>第二节 齐次线性方程组</center></h1>

::: info 例 1
若 $A$ 为 $3\times5$ 矩阵，证明 $A\boldsymbol x=0$ 必有非零解。
:::

::: details 解析
$$
r(A)\le3<5=n,
$$
所以自由变量个数 $n-r(A)>0$，齐次方程组必有非零解。
:::

::: info 例 2
已知齐次方程组有 $4$ 个未知数且 $r(A)=2$，基础解系包含几个向量？
:::

::: details 解析
基础解系含：
$$
n-r(A)=4-2=2
$$
个线性无关解向量。
:::

::: info 例 3
求方程 $x_1+x_2+x_3=0$ 的一个基础解系。
:::

::: details 解析
令 $x_2=s,x_3=t$，则 $x_1=-s-t$：
$$
\boldsymbol x=s(-1,1,0)^T+t(-1,0,1)^T.
$$
一个基础解系为：
$$
(-1,1,0)^T,\quad(-1,0,1)^T.
$$
:::
