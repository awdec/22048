<h1><center>考研数学二-线性代数-习题</center></h1>

## 线代习题目录

这里用于整理考研数学二中线性代数部分的练习题、错题和题型总结。

后续可以按下面模块逐步展开：

- 行列式
- 矩阵
- 向量
- 线性方程组
- 特征值与特征向量
- 二次型
- 综合题

## 题目整理格式

建议每道题按下面结构记录：

1. 题目
2. 思路
3. 解答
4. 关键公式
5. 易错点
6. 同类题总结

## 常见题型标签

整理习题时可以给题目加标签：

- `行列式计算`
- `矩阵方程`
- `矩阵秩`
- `线性相关`
- `基础解系`
- `含参数方程组`
- `特征值`
- `对角化`
- `二次型`
- `正定性`
