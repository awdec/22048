<h1><center>第二节 行列式的性质与展开</center></h1>

::: info 例 1
计算：
$$
D=\begin{vmatrix}1&1&1\\1&2&3\\1&3&6\end{vmatrix}.
$$
:::

::: details 解析
作 $R_2-R_1,R_3-R_1$：
$$
D=\begin{vmatrix}1&1&1\\0&1&2\\0&2&5\end{vmatrix}
=\begin{vmatrix}1&2\\2&5\end{vmatrix}=1.
$$
倍加变换不改变行列式。
:::

::: info 例 2
设三阶矩阵 $A$ 的行列式为 $2$，求 $|-3A^T|$。
:::

::: details 解析
$$
|-3A^T|=(-3)^3|A^T|=-27|A|=-54.
$$
注意三阶矩阵整体乘 $-3$，行列式乘 $(-3)^3$。
:::

::: info 例 3
设 $A=(a_{ij})_{3\times3}$，证明 $a_{11}A_{21}+a_{12}A_{22}+a_{13}A_{23}=0$。
:::

::: details 解析
这是第一行元素与第二行对应代数余子式乘积之和。把第二行替换为第一行后按第二行展开，所得行列式有两行相同，故值为 $0$。
:::
