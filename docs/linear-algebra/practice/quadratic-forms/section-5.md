<h1><center>第六章综合训练与答题技巧</center></h1>

::: info 综合题
设实对称矩阵 $A$ 的特征值为 $1,2,a$，讨论二次型 $\boldsymbol x^TA\boldsymbol x$ 的正定性。
:::

::: details 解析
实对称矩阵正定当且仅当全部特征值为正。已有两个特征值为正，所以：

- $a>0$ 时，二次型正定。
- $a=0$ 时，二次型半正定但不正定。
- $a<0$ 时，既有正特征值又有负特征值，二次型不定。
:::

## 答题技巧

1. 从二次型写矩阵时，交叉项系数除以 $2$。
2. 只问惯性指数时，优先判断特征值符号，不必求特征向量。
3. 二阶、三阶正定题优先用顺序主子式。
4. 要求正交变换时，特征向量必须正交且单位化。
5. 区分相似与合同：前者保持特征值，后者保持惯性指数。
