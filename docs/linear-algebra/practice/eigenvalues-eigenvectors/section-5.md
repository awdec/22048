<h1><center>第五章综合训练与答题技巧</center></h1>

::: info 综合题
设三阶矩阵 $A$ 的特征值为 $1,2,3$，求 $|A^2-4A+5I|$。
:::

::: details 解析
令 $f(t)=t^2-4t+5$，则 $f(A)$ 的特征值为：
$$
f(1)=2,\qquad f(2)=1,\qquad f(3)=2.
$$
所以：
$$
|A^2-4A+5I|=2\cdot1\cdot2=4.
$$
:::

## 答题技巧

1. 先求特征多项式，再求每个特征空间。
2. 用迹与行列式检查特征值结果。
3. 重特征值要比较代数重数和几何重数。
4. 构造 $P$ 或 $Q$ 时，列顺序与对角元顺序保持一致。
5. 实对称矩阵的重特征空间内部要正交化、单位化。
