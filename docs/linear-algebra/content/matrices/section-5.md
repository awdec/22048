<h1><center>第二章总结：矩阵</center></h1>

## 核心公式

| 内容 | 公式 |
| --- | --- |
| 乘积转置 | $(AB)^T=B^TA^T$ |
| 乘积求逆 | $(AB)^{-1}=B^{-1}A^{-1}$ |
| 伴随恒等式 | $AA^*=A^*A=|A|I$ |
| 逆矩阵公式 | $A^{-1}=\dfrac1{|A|}A^*$ |
| 乘积秩 | $r(AB)\le\min\{r(A),r(B)\}$ |
| 可逆保秩 | $r(PAQ)=r(A)$ |

## 可逆的等价条件

对 $n$ 阶方阵 $A$，以下命题等价：

1. $A$ 可逆。
2. $|A|\ne0$。
3. $r(A)=n$。
4. $A$ 可表示为有限个初等矩阵的乘积。
5. $A\boldsymbol x=\boldsymbol0$ 只有零解。
6. $A$ 的行向量组、列向量组都线性无关。

## 高频易错点

- $AB$ 与 $BA$ 可能一个有意义、另一个无意义。
- $(A+B)^{-1}$ 一般不等于 $A^{-1}+B^{-1}$。
- $(AB)^{-1}$、$(AB)^T$ 和 $(AB)^*$ 都要反序。
- $AB=O$ 不能直接推出某个因子为零矩阵。
- 初等变换保秩，但三类行变换对行列式的影响不同。
- 矩阵方程中要根据未知矩阵所在位置决定左乘还是右乘。

::: tip 一句话总结
矩阵题先检查尺寸与可逆性；计算题用初等变换，抽象题用可逆、秩和伴随矩阵之间的等价关系。
:::
