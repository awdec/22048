<h1><center>第一章总结：行列式</center></h1>

## 核心公式

| 内容 | 公式 |
| --- | --- |
| 二阶行列式 | $\begin{vmatrix}a&b\\c&d\end{vmatrix}=ad-bc$ |
| 代数余子式 | $A_{ij}=(-1)^{i+j}M_{ij}$ |
| 按行展开 | $|A|=\sum_j a_{ij}A_{ij}$ |
| 乘积行列式 | $|AB|=|A||B|$ |
| 数乘矩阵 | $|kA|=k^n|A|$ |
| 可逆条件 | $A$ 可逆 $\Longleftrightarrow |A|\ne0$ |
| 伴随矩阵 | $|A^*|=|A|^{n-1}$ |

## 题型与方法

| 题型 | 首选方法 |
| --- | --- |
| 数字型高阶行列式 | 初等变换化三角形 |
| 某行或列零多 | 拉普拉斯展开 |
| 相邻行结构相似 | 逐行相减或递推 |
| 主对角同为 $a$、其余同为 $b$ | 各列相加后提取公因子 |
| 抽象矩阵行列式 | 使用 $|AB|=|A||B|$ |
| 方程组唯一解 | 检查系数行列式是否非零 |

## 必查易错点

1. 交换两行后是否补了负号。
2. 从某一行提出 $k$ 时，只提出一个 $k$。
3. 从整个 $n$ 阶矩阵提出 $k$ 时，行列式变为 $k^n|A|$。
4. 代数余子式的符号是否按 $(-1)^{i+j}$。
5. $|A|=0$ 只表示 $A$ 不可逆，不直接决定非齐次方程组是否有解。

::: tip 一句话总结
行列式计算的目标是“造零与降阶”，判断问题的核心是“行列式非零等价于方阵可逆”。
:::
