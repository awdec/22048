<h1><center>第五章综合训练</center></h1>

::: info 综合题
TCP 发送方从序号 3000 开始连续发送三个各含 1000 字节数据的报文段。第二段丢失，第三段到达接收方。接收方对第三段通常返回什么确认号？
:::

::: details 解析
第一段覆盖序号 3000～3999，接收方期待 4000。第二段 4000～4999 丢失，第三段即使到达也形成空洞。TCP 累计确认通常仍返回：
$$
ACK=4000.
$$
:::

## 答题技巧

- TCP 序号按数据字节数推进，SYN/FIN 额外各加 1。
- 窗口题先取 `min(rwnd, cwnd)`，再减在途未确认数据。
- 握手挥手题逐报文写 SYN、ACK、FIN、seq 和 ack。
