<h1><center>第三节 介质、复用与设备</center></h1>

::: info 例 1
为什么长距离高速骨干链路更常使用单模光纤而非多模光纤？
:::

::: details 解析
单模光纤只有主要传播模式，模间色散小、衰减低，适合高速远距离；多模光纤成本较低，适合较短距离。
:::

::: info 例 2
四个用户采用同步 TDM，共享一条 $8\text{ Mbps}$ 链路，每个用户固定获得相同时隙，单用户平均分配速率是多少？
:::

::: details 解析
忽略控制开销，每个用户固定得到：
$$
8/4=2\text{ Mbps}.
$$
即使某用户无数据，其时隙也可能闲置。
:::

::: info 例 3
集线器的两个端口能否同时无冲突地发送？
:::

::: details 解析
传统共享式集线器的所有端口位于同一冲突域，共享介质，同时发送会冲突。集线器不学习地址，也不定向转发。
:::
