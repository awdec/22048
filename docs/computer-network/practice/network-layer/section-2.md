<h1><center>第二节 IPv4 地址与子网</center></h1>

::: info 例 1
求地址 `192.168.10.130/26` 的网络地址、广播地址和传统可用主机范围。
:::

::: details 解析
/26 每块 64 个地址，130 位于 128～191：

- 网络地址：`192.168.10.128`。
- 广播地址：`192.168.10.191`。
- 可用范围：`192.168.10.129`～`192.168.10.190`。
:::

::: info 例 2
一个 `/27` IPv4 地址块有多少地址和传统可用主机地址？
:::

::: details 解析
$$
2^{32-27}=32
$$
个地址，传统可用主机地址为 $32-2=30$ 个。
:::

::: info 例 3
路由表同时有 `10.0.0.0/8`、`10.1.0.0/16` 和默认路由，目的地址 `10.1.2.3` 选哪一条？
:::

::: details 解析
前两条都匹配，选择最长前缀 `/16`，因为它更具体。
:::

::: info 例 4
将连续的 `192.168.0.0/24` 与 `192.168.1.0/24` 聚合，结果是什么？
:::

::: details 解析
两个连续且对齐的 `/24` 可聚合为：
```text
192.168.0.0/23
```
:::
