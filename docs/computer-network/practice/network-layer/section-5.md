<h1><center>第五节 NAT 与 IPv6</center></h1>

::: info 例 1
两个内网主机使用相同源端口访问同一公网服务器，NAPT 如何区分连接？
:::

::: details 解析
NAPT 为不同内部四元组分配不同公网源端口，维护映射表。返回报文依据公网端口映射回相应内部 IP 和端口。
:::

::: info 例 2
IPv6 基本首部是否包含首部校验和？路由器能否对普通分组分片？
:::

::: details 解析
不包含首部校验和；中间路由器不分片，分片由源主机通过扩展首部完成。
:::

::: info 例 3
IPv6 地址中的 `::` 为什么一个地址只能出现一次？
:::

::: details 解析
`::` 表示一段或多段连续的零。若出现两次，无法唯一判断每处应补多少个零段，地址会产生歧义。
:::

::: info 例 4
移动节点离开归属网络后，通信对端是否必须改用它的转交地址发送分组？
:::

::: details 解析
不必。通信对端仍向归属地址发送；归属代理截获分组并通过隧道转发到已注册的转交地址。这样上层连接使用的归属地址保持不变。
:::
