<h1><center>第一节 IPv4 数据报</center></h1>

::: info 例 1
IPv4 首部长度字段为 6，总长度字段为 1000，求首部和数据长度。
:::

::: details 解析
首部长度单位为 4 字节：
$$
H=6\times4=24\text{ B}.
$$
数据长度：
$$
1000-24=976\text{ B}.
$$
:::

::: info 例 2
一个总长 $4000\text{ B}$、首部 $20\text{ B}$ 的 IPv4 数据报经过 MTU 为 $1500\text{ B}$ 的链路，最大分片数据长度是多少？
:::

::: details 解析
每片最多数据 $1500-20=1480\text{ B}$，且 $1480$ 是 8 的倍数，适合作为非最后分片数据长度。
:::

::: info 例 3
TTL 初值为 64，分组经过 10 台路由器后 TTL 为多少？
:::

::: details 解析
每台路由器至少减 1，若无其他变化则为 $64-10=54$。
:::

::: info 例 4
网络层拥塞控制与传输层流量控制保护的对象有何不同？
:::

::: details 解析
网络层拥塞控制防止网络中的链路、路由器和缓存过载，保护网络整体；流量控制限制发送速度以适应接收端处理和缓存能力，主要保护接收端。
:::
