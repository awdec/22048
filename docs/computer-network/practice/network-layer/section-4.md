<h1><center>第四节 路由算法与协议</center></h1>

::: info 例 1
路由器 X 到邻居 V 的代价为 2，V 通告到目的 Y 的距离为 5，则经 V 到 Y 的候选距离是多少？
:::

::: details 解析
由距离向量更新：
$$
c(X,V)+D_V(Y)=2+5=7.
$$
再与其他邻居候选距离取最小值。
:::

::: info 例 2
RIP 中到某网络的跳数为 16 表示什么？
:::

::: details 解析
表示不可达。RIP 最大有效跳数为 15，因此只适合规模较小的网络。
:::

::: info 例 3
OSPF 路由器是否只把路由表发给邻居？
:::

::: details 解析
不是。OSPF 发现本地链路状态后将链路状态信息在区域内可靠洪泛，各路由器建立一致的链路状态数据库，再独立运行 Dijkstra 算法。
:::

::: info 例 4
BGP 为什么不能简单理解为“选择跳数最少的路径”？
:::

::: details 解析
BGP 是跨自治系统的路径向量协议，选择需考虑本地策略、AS 路径、下一跳等属性，可扩展性和管理策略常比物理最短路径更重要。
:::

::: info 例 5
层次路由为什么能减小路由信息规模？
:::

::: details 解析
路由器只需详细掌握本区域或本自治系统内部的路由，对外部网络主要保存聚合后的可达信息。这样可减少路由表项和路由更新范围，同时允许不同管理域采用不同策略。
:::
