<h1><center>第四章综合训练</center></h1>

::: info 综合题
学校获得地址块 `10.0.0.0/24`，要划分至少 6 个等长子网，每个子网至少可容纳 25 台主机。选择前缀长度并说明容量。
:::

::: details 解析
至少 6 个子网需借 3 位，得到 $2^3=8$ 个子网，前缀为 `/27`。每个子网有 $2^5=32$ 个地址，传统可用主机数 $30$，满足至少 25 台。
:::

## 答题技巧

- 子网题先从主机数确定主机位，再检查子网数量。
- 分片题中片偏移按 8 字节计，只有最后一片数据长度可不为 8 的倍数。
- 跨网封装固定区分最终 IP 与下一跳 MAC。

::: info 综合题 2
说明路由表与转发表的区别，并指出输入端口收到分组后主要查询哪一个。
:::

::: details 解析
路由表属于控制平面，保存路由协议学习到的网络可达信息；转发表由路由表生成，属于数据平面，保存目的前缀对应的下一跳和输出接口。输入端口转发分组时主要查询转发表。
:::

::: info 综合题 3
SDN 为什么被概括为“控制平面与数据平面分离”？
:::

::: details 解析
逻辑集中的控制器掌握网络状态、计算路径并下发转发规则；交换设备的数据平面按规则高速处理分组，不必在每台设备中分别完成全部控制决策。
:::
