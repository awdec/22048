<h1><center>第一章综合训练</center></h1>

::: info 综合题
主机通过两台存储转发路由器发送一个长度为 $12000$ bit 的分组，三条链路速率均为 $2\text{ Mbps}$，忽略传播、排队和处理时延。求分组从开始发送到完整到达的时间。
:::

::: details 解析
一个分组经过三条链路，每条链路发送时延：
$$
\frac{12000}{2\times10^6}=6\text{ ms}.
$$
存储转发要求完整接收后再发送，总时间：
$$
3\times6=18\text{ ms}.
$$
:::

## 答题技巧

- 时延题先拆处理、排队、发送和传播。
- 多跳存储转发题明确分组数和链路数。
- 设备题先判断它检查哪一层首部或地址。
