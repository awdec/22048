<h1><center>第三节 分层与协议</center></h1>

::: info 例 1
“规定确认报文中各字段的含义”属于协议三要素中的哪一个？
:::

::: details 解析
属于语义。数据字段格式属于语法，发送和响应的先后次序属于同步。
:::

::: info 例 2
交换机根据 MAC 地址转发帧，路由器根据 IP 地址转发分组，它们主要工作在哪一层？
:::

::: details 解析
二层交换机主要工作在数据链路层；路由器主要工作在网络层。
:::

::: info 例 3
应用报文从发送主机到接收主机时，封装和解封装方向如何？
:::

::: details 解析
发送端从应用层向下逐层添加首部（链路层还可能加尾部）；接收端从物理层向上逐层检查并去除相应控制信息。
:::
