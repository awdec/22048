<h1><center>第一节 网络组成与分类</center></h1>

::: info 例 1
路由器属于互联网的边缘部分还是核心部分？
:::

::: details 解析
通常属于核心部分，负责在不同链路和网络之间转发分组。端系统及其应用构成边缘部分。
:::

::: info 例 2
比较电路交换与分组交换对突发数据的适应性。
:::

::: details 解析
电路交换需先建立并独占资源，突发间歇期间资源可能闲置；分组交换统计复用链路，无需为单个通信长期预留资源，更适合突发数据。
:::

::: info 例 3
星形网络中一条主机链路故障和中心节点故障的影响有何不同？
:::

::: details 解析
单条主机链路故障通常只影响该主机；中心节点故障可能导致整个星形网络无法通信，因此中心节点是单点故障。
:::
