<h1><center>第四节 电子邮件与 FTP</center></h1>

::: info 例 1
用户从自己的邮件客户端向邮件服务器发送邮件，以及从服务器读取邮件，分别常用什么协议？
:::

::: details 解析
发送和服务器之间转发使用 SMTP；用户从服务器读取邮件使用 POP3。
:::

::: info 例 2
FTP 为什么使用两条 TCP 连接？
:::

::: details 解析
控制连接长期传输命令和响应，数据连接按文件或目录传输任务建立并关闭，使控制与数据相互独立。
:::

::: info 例 3
为什么 FTP 被动模式更容易穿越客户端侧 NAT 和防火墙？
:::

::: details 解析
被动模式的数据连接也由客户端主动发起，NAT 和防火墙通常允许内部发起的出站连接；主动模式要求服务器反向连接客户端端口，容易被拦截。
:::

::: info 例 4
MIME 的主要作用是什么？
:::

::: details 解析
为电子邮件定义内容类型和传输编码，使原本面向 ASCII 文本的邮件系统可以传输非 ASCII 文本、图片、音视频和附件。
:::
