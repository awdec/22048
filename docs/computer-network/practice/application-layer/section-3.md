<h1><center>第三节 HTTP</center></h1>

::: info 例 1
忽略 DNS 和对象传输时间，使用非持续 HTTP 获取一个对象至少约需几个 RTT？
:::

::: details 解析
约 $2RTT$：一个 RTT 建立 TCP 连接，一个 RTT 发送 HTTP 请求并收到响应首字节。
:::

::: info 例 2
HTTP 是无状态协议，为什么网站仍能保持登录状态？
:::

::: details 解析
可通过 Cookie、会话标识或令牌把连续请求关联起来。无状态描述的是 HTTP 核心请求—响应协议不自动保存应用会话。
:::

::: info 例 3
缓存向服务器发送条件请求，资源未修改时典型返回什么状态码？
:::

::: details 解析
返回 `304 Not Modified`，客户端继续使用缓存副本，无需重新传输完整对象。
:::

::: info 例 4
HTTP 请求报文和响应报文的首行分别叫什么，主要包含什么？
:::

::: details 解析
请求报文首行称为请求行，主要包含方法、资源路径和 HTTP 版本；响应报文首行称为状态行，主要包含 HTTP 版本、状态码和原因短语。
:::
