<h1><center>第四节 局域网、无线局域网、广域网与 PPP</center></h1>

::: info 例 1
交换机端口 1 收到源 MAC 为 A、目的 MAC 为 B 的帧，表中没有 A 和 B。交换机如何处理？
:::

::: details 解析
先学习 `A → 端口1`，再因目的 B 未知，向除端口 1 外的同 VLAN 端口泛洪。
:::

::: info 例 2
交换机表中 B 也位于入端口 1，收到发往 B 的帧后如何处理？
:::

::: details 解析
过滤该帧，不向其他端口转发，因为源和目的位于同一端口方向。
:::

::: info 例 3
一个 8 端口二层交换机所有端口属于同一 VLAN，共有多少冲突域和广播域？
:::

::: details 解析
每个端口独立冲突域，共 8 个冲突域；同一 VLAN 默认是一个广播域。
:::

::: info 例 4
基础设施模式的 IEEE 802.11 无线局域网中，AP 的作用是什么？
:::

::: details 解析
AP 组织一个基本服务集 BSS，供无线站点关联，并在无线站点之间或无线站点与分配系统之间转发帧。无线站点通常先与 AP 建立关联，再通过 AP 接入其他网络。
:::

::: info 例 5
PPP 是否使用确认和重传实现可靠传输？
:::

::: details 解析
不使用。PPP 用 FCS 检测帧差错，但差错帧直接丢弃，不提供序号、确认、重传和流量控制；需要可靠性时由上层协议负责。
:::
