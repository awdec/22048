<h1><center>第一节 成帧与差错检测</center></h1>

::: info 例 1
比特填充中，数据为 `01111110`，填充后的数据是什么？
:::

::: details 解析
发送方每出现连续 5 个 `1` 就插入一个 `0`：
```text
011111010
```
接收方见到连续 5 个 `1` 后的 `0` 会将其删除。
:::

::: info 例 2
数据为 `1101`，生成多项式比特串为 `1011`，CRC 计算时应先补几个 0？余数多少位？
:::

::: details 解析
生成串长度为 4，最高次数为 3，因此数据后补 3 个 `0`，最终余数为 3 位。
:::

::: info 例 3
需要传输 $k=8$ 个信息位，使用可纠正一位错误的海明码，至少需要多少校验位？
:::

::: details 解析
要求：
$$
2^r\ge k+r+1.
$$
$r=3$ 时 $8<12$ 不满足；$r=4$ 时 $16\ge13$，故至少 4 位。
:::
