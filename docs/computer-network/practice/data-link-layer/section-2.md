<h1><center>第二节 流量控制与可靠传输</center></h1>

::: info 例 1
停止—等待协议中，数据帧发送时延 $1\text{ ms}$，单程传播时延 $9\text{ ms}$，忽略确认发送时间，求信道利用率。
:::

::: details 解析
$$
U=\frac1{1+2\times9}=\frac1{19}\approx5.26\%.
$$
:::

::: info 例 2
GBN 使用 3 位序号，发送窗口最大是多少？
:::

::: details 解析
$$
W_T\le2^3-1=7.
$$
接收窗口为 1。
:::

::: info 例 3
SR 使用 4 位序号且发送、接收窗口相等，窗口最大是多少？
:::

::: details 解析
为了避免新旧帧序号混淆：
$$
W_T=W_R\le2^{4-1}=8.
$$
:::

::: info 例 4
GBN 中帧 3 丢失，而帧 4、5 已发送，超时后重传哪些帧？
:::

::: details 解析
接收方不缓存失序帧，发送方重传帧 3 及之后所有未确认帧，即 3、4、5。
:::
