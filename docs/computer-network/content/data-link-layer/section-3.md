<h1><center>第三节 介质访问控制</center></h1>

## 1. 信道划分协议

- FDM、TDM、WDM、CDM：静态划分资源，冲突少，低负载时可能浪费。

## 2. 随机访问协议

### 2.1 ALOHA

纯 ALOHA 站点有帧即发送，冲突后随机重传；时隙 ALOHA 只能在时隙开始发送，缩小冲突时间窗口。

### 2.2 CSMA

发送前先监听载波：

- 1-坚持：空闲立即发送，忙则持续监听。
- 非坚持：忙则随机等待后再监听。
- $p$-坚持：时隙信道空闲时以概率 $p$ 发送。

### 2.3 CSMA/CD

载波监听多路访问/冲突检测用于传统半双工共享以太网。发送时继续检测冲突，发现冲突后发送强化信号并执行二进制指数退避。

争用期为往返传播时延：

$$
2\tau.
$$

为保证发送方在帧发送完前检测到最远端冲突，最短帧发送时间需满足：

$$
T_{frame}\ge2\tau.
$$

### 2.4 CSMA/CA

无线局域网难以边发送边检测冲突，因此采用冲突避免：帧间间隔、随机退避、确认帧和可选 RTS/CTS。

## 3. 轮询访问

- 轮询协议：主节点依次询问从节点，控制开销和单点故障明显。
- 令牌传递：只有持有令牌的站点可发送，无冲突，重负载下性能稳定。

## 4. 隐蔽站与暴露站

- 隐蔽站：两个发送站彼此听不到，却会在共同接收端冲突。
- 暴露站：某站听到邻站发送而错误推迟，虽然向另一个方向发送本可成功。

RTS/CTS 可缓解隐蔽站问题。

::: warning 易错点
全双工交换式以太网不存在共享介质冲突，不使用 CSMA/CD；无线局域网采用 CSMA/CA，不能可靠地直接检测冲突。
:::
