<h1><center>第三节 传输介质、复用与物理层设备</center></h1>

## 1. 导向传输介质

### 1.1 双绞线

成本低、应用广，可分为屏蔽双绞线 STP 和非屏蔽双绞线 UTP。

### 1.2 同轴电缆

抗干扰能力较强，曾广泛用于有线电视和早期以太网。

### 1.3 光纤

利用全反射传播光信号，带宽大、衰减小、抗电磁干扰、安全性高。

- 多模光纤：多条光路，适合较短距离。
- 单模光纤：单一路径，适合高速远距离。

## 2. 非导向介质

包括无线电波、微波、红外线和可见光。无线电波穿透能力较强；微波通常视距传播，可用于地面中继和卫星通信。

## 3. 信道复用

### 3.1 频分复用 FDM

不同用户占用不同频带，同时传输，频带之间设置保护间隔。

### 3.2 时分复用 TDM

不同用户周期性占用不同时间片。同步 TDM 即使用户无数据也可能保留时隙；统计 TDM 动态分配时隙。

### 3.3 波分复用 WDM

在光纤中用不同光波长并行传输，本质上是光域的频分复用。

### 3.4 码分复用 CDM

用户在相同时间、相同频带发送，通过相互正交的码片序列区分。接收端用内积恢复目标用户数据。

## 4. 中继器

中继器工作在物理层，对衰减或失真的数字信号进行再生与转发，不识别帧、MAC 地址或 IP 地址。

## 5. 集线器

集线器是多端口中继器，收到比特后向其他端口广播。所有端口属于同一冲突域，通常共享带宽，不能隔离广播域。

::: warning 易错点
中继器和集线器只扩展物理覆盖范围，不能连接不同链路层协议，也不能根据地址过滤流量。
:::
