<h1><center>第五章总结：传输层</center></h1>

## TCP 与 UDP

| 项目 | TCP | UDP |
| --- | --- | --- |
| 连接 | 面向连接 | 无连接 |
| 可靠性 | 可靠、按序 | 尽最大努力 |
| 数据边界 | 字节流 | 保留报文边界 |
| 首部 | 至少 20 B | 8 B |
| 通信方式 | 一对一连接 | 支持多种方式 |

## TCP 核心计算

- 确认号 = 首字节序号 + 数据长度。
- SYN/FIN 各消耗 1 个序号。
- 实际窗口 = $\min(rwnd,cwnd)$。
- 典型 IPv4/TCP MSS = MTU − 20 − 20。

::: tip 一句话总结
传输层用端口完成进程到进程交付；UDP追求简单低开销，TCP用序号、确认、窗口和计时器实现可靠字节流。
:::
