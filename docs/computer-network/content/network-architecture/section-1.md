<h1><center>第一节 网络概念、组成与分类</center></h1>

## 1. 计算机网络

计算机网络是将分散、具有独立功能的计算机系统，通过通信设备与线路连接起来，由功能完善的软件实现资源共享和信息传递的系统。

基本功能包括：

- 数据通信。
- 资源共享。
- 分布式处理。
- 提高可靠性与负载均衡。

## 2. 网络组成

从组成部分看：

- 硬件：主机、链路、交换设备、路由器等。
- 软件：网络操作系统、应用和协议实现。
- 协议：规定通信双方交换信息的规则。

从工作方式看：

- 边缘部分：端系统及运行其上的应用。
- 核心部分：路由器和通信链路，负责分组交换。

## 3. 网络分类

### 3.1 按覆盖范围

- 个域网 PAN。
- 局域网 LAN。
- 城域网 MAN。
- 广域网 WAN。

### 3.2 按使用者

- 公用网。
- 专用网。

### 3.3 按传输技术

- 广播式网络：共享通信信道。
- 点对点网络：节点之间通过逐跳链路转发。

## 4. 拓扑结构

| 拓扑 | 优点 | 缺点 |
| --- | --- | --- |
| 总线 | 结构简单、成本低 | 故障定位困难、冲突多 |
| 星形 | 易管理、单链路故障影响小 | 中心节点是瓶颈 |
| 环形 | 控制简单、传输有序 | 单点故障可能影响全环 |
| 网状 | 可靠、路径多 | 线路多、成本高 |

## 5. 交换技术

- **电路交换**：建立连接、通信、释放连接；独占资源，适合连续数据流。
- **报文交换**：整份报文存储转发，无需预建连接，缓存需求大。
- **分组交换**：报文切分为分组逐跳存储转发，链路利用率高，是互联网核心方式。

::: warning 易错点
互联网不是某一种物理网络，而是通过 TCP/IP 协议族把大量异构网络互联形成的网络之网。
:::
