<h1><center>第四节 电子邮件与 FTP</center></h1>

## 1. 电子邮件系统

组成包括用户代理、邮件服务器和邮件传输协议。

- SMTP：发送与服务器之间转发邮件，使用 TCP，常见服务器端口 25。
- POP3：客户端下载邮件，功能简单，常见端口 110。

邮件发送通常使用 SMTP，接收端用户从邮件服务器读取邮件使用 POP3。

## 2. 电子邮件格式与 MIME

电子邮件由首部和主体组成，二者之间用空行分隔。首部通常包含 `From`、`To`、`Subject`、`Date` 等字段；主体保存邮件正文。

传统 SMTP 主要处理 ASCII 文本。MIME 增加内容类型、内容传输编码等首部，使邮件可携带非 ASCII 文本、图片、音频和附件。

## 3. FTP

FTP 使用两条 TCP 连接：

- 控制连接：服务器端口 21，整个会话期间保持。
- 数据连接：每次传输文件或目录时建立，传输后关闭。

主动模式下服务器常从端口 20 主动连接客户端；被动模式下客户端连接服务器告知的数据端口，更适合穿越 NAT 和防火墙。

## 4. 常见协议端口

| 协议 | 传输层 | 默认端口 |
| --- | --- | --- |
| FTP 控制 | TCP | 21 |
| SMTP | TCP | 25 |
| DNS | UDP/TCP | 53 |
| HTTP | TCP | 80 |
| POP3 | TCP | 110 |

::: warning 易错点
FTP 的控制连接与数据连接相互独立；SMTP 用于推送邮件，POP3 用于用户读取邮件。
:::
