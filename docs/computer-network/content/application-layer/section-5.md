<h1><center>第六章总结：应用层</center></h1>

## 协议速查

| 协议 | 主要功能 | 传输层/端口 |
| --- | --- | --- |
| DNS | 域名解析 | UDP/TCP 53 |
| HTTP | Web 请求响应 | TCP 80 |
| SMTP | 发送邮件 | TCP 25 |
| POP3 | 下载邮件 | TCP 110 |
| FTP | 文件传输 | TCP 21 控制连接 |

## 高频对比

- C/S 管理集中，P2P 自扩展性强。
- DNS 递归查询由服务器继续查询，迭代查询返回下一步线索。
- HTTP 无状态不等于非持续连接。
- SMTP 推送邮件，POP3 从服务器读取邮件。
- FTP 控制连接持久，数据连接按传输任务建立。

::: tip 一句话总结
应用层协议定义具体业务报文与交互规则，并借助传输层的端口、可靠性或低时延服务完成进程通信。
:::
