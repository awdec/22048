<h1><center>第二节 计算机的性能指标</center></h1>

## 1. 性能指标的作用

计算机性能指标用于描述计算机系统的运行速度、处理能力、存储能力和数据传输能力。

408 中性能指标常以计算题形式出现，重点不是背名词，而是会使用公式分析程序执行时间。

常见性能指标包括：

- 机器字长
- 主频
- 时钟周期
- CPI
- 程序执行时间
- MIPS
- FLOPS
- 吞吐量
- 响应时间
- 存储容量
- 带宽

## 2. 机器字长

机器字长是指计算机一次整数运算所能处理的二进制位数，通常与 CPU 内部寄存器位数和 ALU 位数有关。

常见机器字长：

- $8$ 位
- $16$ 位
- $32$ 位
- $64$ 位

机器字长越长，通常一次能处理的数据范围越大，但不能简单认为机器字长越长，计算机速度一定越快。

容易混淆的几个概念：

| 概念 | 含义 |
| --- | --- |
| 机器字长 | CPU 一次整数运算能处理的二进制位数 |
| 指令字长 | 一条指令包含的二进制位数 |
| 存储字长 | 一个存储单元存放的二进制位数 |
| 数据通路宽度 | 数据在部件之间一次传输的位数 |

::: warning 注意
机器字长、指令字长、存储字长不一定相等。题目若没有明确说明，不能默认三者相同。
:::

## 3. 主频与时钟周期

计算机中许多操作都受到时钟信号控制。

时钟周期是 CPU 时钟脉冲的周期，主频是单位时间内时钟周期的次数。

二者关系为：

$$
f = \frac{1}{T}
$$

其中：

- $f$ 表示主频。
- $T$ 表示时钟周期。

若主频为 $1\text{ GHz}$，则时钟周期为：

$$
T = \frac{1}{10^9}\text{ s} = 1\text{ ns}
$$

常见单位换算：

| 单位 | 含义 |
| --- | --- |
| $1\text{ ms}$ | $10^{-3}\text{ s}$ |
| $1\text{ us}$ | $10^{-6}\text{ s}$ |
| $1\text{ ns}$ | $10^{-9}\text{ s}$ |
| $1\text{ ps}$ | $10^{-12}\text{ s}$ |

常见频率单位：

| 单位 | 换算 |
| --- | --- |
| $1\text{ kHz}$ | $10^3\text{ Hz}$ |
| $1\text{ MHz}$ | $10^6\text{ Hz}$ |
| $1\text{ GHz}$ | $10^9\text{ Hz}$ |
| $1\text{ THz}$ | $10^{12}\text{ Hz}$ |
| $1\text{ PHz}$ | $10^{15}\text{ Hz}$ |

## 4. CPI

CPI 是 Cycles Per Instruction 的缩写，表示平均每条指令需要的时钟周期数。

定义为：

$$
CPI = \frac{\text{执行全部指令所需时钟周期数}}{\text{指令条数}}
$$

如果不同类型指令的 CPI 不同，则平均 CPI 应按指令比例加权计算：

$$
CPI = \sum_{i=1}^{n} p_i \cdot CPI_i
$$

其中：

- $p_i$ 表示第 $i$ 类指令所占比例。
- $CPI_i$ 表示第 $i$ 类指令的 CPI。

## 5. 程序执行时间

程序执行时间是性能计算中最重要的指标。

常用公式：

$$
\text{执行时间}
=
\text{指令条数}
\times
CPI
\times
\text{时钟周期}
$$

也可以写成：

$$
\text{执行时间}
=
\frac{\text{指令条数} \times CPI}{\text{主频}}
$$

记作：

$$
T
=
\frac{IC \times CPI}{f}
$$

其中：

- $T$ 表示程序执行时间。
- $IC$ 表示指令条数。
- $CPI$ 表示平均每条指令需要的时钟周期数。
- $f$ 表示主频。

这个公式说明，程序执行时间受到三个因素影响：

| 因素 | 说明 |
| --- | --- |
| 指令条数 | 与算法、编译器、指令系统有关 |
| CPI | 与指令类型、CPU 结构、流水线、Cache 等有关 |
| 主频 | 与硬件工艺和时钟设计有关 |

因此，主频高不一定执行时间短，还要看指令条数和 CPI。

## 6. MIPS

MIPS 表示每秒执行百万条指令数。

$$
MIPS
=
\frac{\text{指令条数}}{\text{执行时间} \times 10^6}
$$

由程序执行时间公式可得：

$$
MIPS
=
\frac{f}{CPI \times 10^6}
$$

MIPS 适合粗略描述指令执行速度，但有明显局限：

- 不同计算机的指令系统可能不同。
- 不同指令完成的工作量不同。
- MIPS 高不一定表示实际程序执行更快。

## 7. FLOPS

FLOPS 表示每秒浮点运算次数，常用于衡量浮点计算能力。

常见单位：

| 单位 | 含义 |
| --- | --- |
| KFLOPS | 每秒 $10^3$ 次浮点运算 |
| MFLOPS | 每秒 $10^6$ 次浮点运算 |
| GFLOPS | 每秒 $10^9$ 次浮点运算 |
| TFLOPS | 每秒 $10^{12}$ 次浮点运算 |
| PFLOPS | 每秒 $10^{15}$ 次浮点运算 |
| EFLOPS | 每秒 $10^{18}$ 次浮点运算 |

计算公式：

$$
FLOPS
=
\frac{\text{浮点运算次数}}{\text{执行时间}}
$$

MIPS 关注指令条数，FLOPS 关注浮点运算次数，二者不能直接等同。

## 8. 吞吐量与响应时间

吞吐量表示单位时间内系统完成的任务数量。

响应时间表示从用户提交任务到系统完成该任务所需的时间。

| 指标 | 关注点 | 例子 |
| --- | --- | --- |
| 吞吐量 | 单位时间完成多少任务 | 每秒处理多少个请求 |
| 响应时间 | 单个任务多久完成 | 打开一个程序需要多久 |

提高吞吐量不一定会降低响应时间。比如批处理系统可以提高单位时间处理任务数，但单个任务等待时间可能变长。

## 9. 存储容量与带宽

存储容量表示存储器能够保存的信息量。

常见单位：

| 单位 | 换算 |
| --- | --- |
| $1\text{ B}$ | $8\text{ bit}$ |
| $1\text{ KB}$ | $2^{10}\text{ B}$ |
| $1\text{ MB}$ | $2^{20}\text{ B}$ |
| $1\text{ GB}$ | $2^{30}\text{ B}$ |
| $1\text{ TB}$ | $2^{40}\text{ B}$ |
| $1\text{ PB}$ | $2^{50}\text{ B}$ |
| $1\text{ EB}$ | $2^{60}\text{ B}$ |

容量单位相邻两级通常相差：

$$
2^{10} = 1024
$$

带宽表示单位时间内传输的数据量。

$$
\text{带宽}
=
\frac{\text{传输的数据量}}{\text{传输时间}}
$$

408 中常见带宽单位包括：

- $\text{bit/s}$
- $\text{B/s}$
- $\text{MB/s}$
- $\text{GB/s}$
- $\text{TB/s}$
- $\text{PB/s}$

注意区分 bit 和 Byte：

$$
1\text{ B} = 8\text{ bit}
$$

还要注意单位换算采用的进制：

| 场景 | 常见换算 |
| --- | --- |
| 主频、FLOPS、MIPS | 通常按 $10^3$ 逐级换算 |
| 存储容量 | 计组题中通常按 $2^{10}$ 逐级换算 |
| 数据传输速率 | 题目若写 $\text{MB/s}$、$\text{GB/s}$，需按题目约定判断 |

## 10. 性能提升与加速比

比较两个系统或两种方案的性能时，常用加速比。

$$
\text{加速比}
=
\frac{\text{改进前执行时间}}{\text{改进后执行时间}}
$$

如果加速比大于 $1$，说明改进后更快。

例如，原来执行时间为 $10\text{ s}$，改进后为 $5\text{ s}$，则：

$$
\text{加速比}
=
\frac{10}{5}
=
2
$$

表示性能提高为原来的 $2$ 倍。

## 11. 408 常见考法

| 考法 | 示例 |
| --- | --- |
| 主频换算 | 已知主频求时钟周期 |
| CPI 计算 | 根据不同指令比例求平均 CPI |
| 执行时间计算 | 使用 $T = \frac{IC \times CPI}{f}$ |
| MIPS 计算 | 根据主频和 CPI 求 MIPS |
| 单位换算 | bit、B、KB、MB、GB、TB、PB 之间换算 |
| 指标辨析 | 判断主频、CPI、吞吐量、响应时间的含义 |

## 12. 易错点

- 主频越高，计算机不一定越快。
- CPI 是平均每条指令的时钟周期数，不是执行一条指令的固定周期数。
- 指令条数、CPI、主频共同决定程序执行时间。
- MIPS 不能可靠比较不同指令系统计算机的实际性能。
- bit 和 Byte 不要混淆，$1\text{ B} = 8\text{ bit}$。
- 计组中存储容量通常按 $2^{10}$ 换算，除非题目特别说明。
- 主频、FLOPS 等性能单位通常按 $10^3$ 换算，不要和存储容量的 $2^{10}$ 混淆。
- 吞吐量和响应时间关注角度不同，不能混为一谈。

## 13. 本节小结

计算机性能指标的核心公式是：

$$
T
=
\frac{IC \times CPI}{f}
$$

复习时要围绕这个公式理解：

- 程序有多少条指令。
- 每条指令平均需要多少周期。
- 每秒有多少个时钟周期。

只背“主频越高越快”是远远不够的。408 更常考的是能否根据题目条件，把指令条数、CPI、主频和执行时间之间的关系算清楚。
