<h1><center>第二节 主存储器</center></h1>

## 1. 主存储器的作用

主存储器简称主存，也称内存，用于存放当前正在运行的程序和数据。

CPU 执行程序时，需要从主存中取出指令，也需要从主存中读写数据。

主存的特点：

- 可被 CPU 直接访问。
- 容量大于 Cache，小于外存。
- 速度慢于 Cache，快于外存。
- 通常由半导体存储器构成。

## 2. 主存储器的基本组成

主存通常由存储体、地址译码器、读写控制电路等部分组成。

| 部件 | 作用 |
| --- | --- |
| 存储体 | 保存二进制信息 |
| 地址译码器 | 根据地址选择对应存储单元 |
| 读写控制电路 | 控制读操作和写操作 |
| 数据缓冲 | 暂存输入或输出数据 |

从 CPU 角度看，主存主要通过地址、数据和控制信号进行访问。

## 3. 存储单元、存储字和存储字长

存储单元是 CPU 可访问的最小地址单位。

存储字是一个存储单元中保存的一组二进制位。

存储字长是一个存储字包含的二进制位数。

例如，某存储器容量为：

$$
1K \times 8\text{ bit}
$$

表示该存储器有：

$$
1K = 1024
$$

个存储单元，每个存储单元为 $8$ 位。

总容量为：

$$
1024 \times 8\text{ bit}
$$

也就是：

$$
1024\text{ B}
$$

## 4. SRAM

SRAM 是静态随机存取存储器。

SRAM 使用触发器保存信息，只要不断电，信息就可以保持，不需要刷新。

特点：

- 速度快。
- 不需要刷新。
- 集成度低。
- 成本高。
- 常用于 Cache。

SRAM 的一个存储单元通常由多个晶体管组成，因此面积较大，但访问速度快。

## 5. DRAM

DRAM 是动态随机存取存储器。

DRAM 使用电容存储电荷表示信息。由于电容电荷会泄漏，因此需要周期性刷新。

特点：

- 集成度高。
- 容量大。
- 成本低。
- 速度慢于 SRAM。
- 需要刷新。
- 常用于主存。

SRAM 与 DRAM 对比：

| 项目 | SRAM | DRAM |
| --- | --- | --- |
| 存储元件 | 触发器 | 电容 |
| 是否刷新 | 不需要 | 需要 |
| 速度 | 快 | 较慢 |
| 集成度 | 低 | 高 |
| 成本 | 高 | 低 |
| 常见用途 | Cache | 主存 |

## 6. DRAM 刷新

DRAM 中电容电荷会逐渐泄漏，因此必须在规定时间内重新写入信息，这个过程称为刷新。

刷新特点：

- 刷新以行为单位进行。
- 刷新由存储器控制电路完成。
- 刷新期间可能影响正常读写。

常见刷新方式：

| 刷新方式 | 特点 |
| --- | --- |
| 集中刷新 | 在一段时间内集中刷新所有行，存在较长死区 |
| 分散刷新 | 每个存取周期后安排一次刷新，速度受影响较稳定 |
| 异步刷新 | 在刷新周期内均匀安排各行刷新，折中较好 |

设 DRAM 有 $R$ 行，刷新周期为 $T$，则平均每隔：

$$
\frac{T}{R}
$$

就需要刷新一行。

## 7. ROM 和 Flash

ROM 是只读存储器，断电后信息不丢失。

常见 ROM 类型：

| 类型 | 特点 |
| --- | --- |
| Mask ROM | 出厂时写入，之后不可修改 |
| PROM | 用户可写一次 |
| EPROM | 可用紫外线擦除后重写 |
| EEPROM | 可电擦除和重写 |
| Flash | 可按块擦除，广泛用于固态存储 |

Flash 具有非易失性，常用于 U 盘、SSD、嵌入式系统存储等。

## 8. 多字节数据存放方式

当一个数据占多个字节时，需要确定这些字节在内存中的排列顺序。

常见方式：

| 方式 | 含义 |
| --- | --- |
| 大端方式 | 数据高位字节存放在低地址 |
| 小端方式 | 数据低位字节存放在低地址 |

例如，32 位数据：

$$
0x12345678
$$

若从地址 $A$ 开始存放：

| 地址 | 大端方式 | 小端方式 |
| --- | --- | --- |
| $A$ | $12$ | $78$ |
| $A+1$ | $34$ | $56$ |
| $A+2$ | $56$ | $34$ |
| $A+3$ | $78$ | $12$ |

408 中若涉及多字节数据存储，要看清题目采用大端还是小端。

## 9. 主存容量计算

若主存地址线有 $n$ 根，且按字节编址，则可寻址容量为：

$$
2^n\text{ B}
$$

若按字编址，每个字为 $W$ 位，则可寻址容量为：

$$
2^n \times W\text{ bit}
$$

存储芯片常写作：

$$
M \times W
$$

其中：

- $M$ 表示存储单元个数。
- $W$ 表示每个存储单元位数。

例如：

$$
4K \times 8
$$

表示有 $4K$ 个单元，每个单元 $8$ 位。

## 10. 408 常见考法

| 考法 | 示例 |
| --- | --- |
| SRAM 与 DRAM 对比 | 判断谁需要刷新、谁速度更快 |
| DRAM 刷新 | 根据行数和刷新周期计算刷新间隔 |
| 容量计算 | 根据芯片规格求总容量 |
| 编址方式 | 判断按字节编址和按字编址下的地址范围 |
| 端序判断 | 判断多字节数据在内存中的排列 |

## 11. 易错点

- DRAM 是随机存取存储器，但需要刷新。
- SRAM 不需要刷新，但断电后信息仍会丢失。
- ROM、Flash 是非易失性存储器。
- 存储芯片规格中的 $K$ 通常表示 $2^{10}$。
- 大端和小端只影响多字节数据在内存中的字节顺序，不改变数据本身的数值。

## 12. 本节小结

主存是 CPU 可以直接访问的大容量工作存储器。408 中主存部分重点掌握 SRAM、DRAM、刷新、容量计算和多字节数据存放方式，为后续主存连接、Cache 和虚拟存储器打基础。
