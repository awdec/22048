<h1><center>第六章 总线</center></h1>

## 本章定位

总线是连接 CPU、主存储器和 I/O 设备的信息通路。

前面几章分别讨论了 CPU、存储器和 I/O 系统中的部件，本章关注这些部件之间如何交换地址、数据和控制信息。

按照 2026 年考研 408 计算机组成原理的复习要求，总线部分主要把握两条主线：

1. 总线是什么、由哪些信号线组成、如何分类、如何评价性能。
2. 一次总线传输如何发生，定时方式如何保证主设备和从设备协调工作。

## 学习主线

总线章节可以按下面的逻辑理解：

$$
\text{部件互连}
\rightarrow
\text{共享通路}
\rightarrow
\text{总线仲裁}
\rightarrow
\text{总线事务}
\rightarrow
\text{定时控制}
\rightarrow
\text{性能计算}
$$

也可以记成一句话：

> 多个部件共用一组线路时，必须先解决“谁用总线”，再解决“怎么传数据”，最后解决“传得多快”。

## 408 常见考点

| 考点 | 要求 |
| --- | --- |
| 总线组成 | 掌握地址总线、数据总线、控制总线的作用 |
| 总线分类 | 区分片内总线、系统总线、通信总线 |
| 总线性能 | 会计算总线带宽、总传输时间 |
| 总线仲裁 | 理解菊花链、计数器查询、独立请求方式 |
| 总线事务 | 会分析读事务、写事务、DMA 事务的基本流程 |
| 总线定时 | 比较同步、异步、半同步、分离式通信 |

## 易错点

- 数据总线宽度影响一次能传多少数据，地址总线宽度影响可寻址空间大小。
- 总线周期不等于时钟周期。一个总线周期通常包含多个时钟周期。
- 总线带宽不仅由数据线宽度决定，还与总线频率、传输次数和协议开销有关。
- 同步定时不是一定最快，若连接慢速设备，可能需要插入等待周期。
- 异步定时适合速度差异较大的设备，但握手控制较复杂。
- 分离式通信不是把一次通信取消，而是把请求和响应分开，占用总线的时间更短。
