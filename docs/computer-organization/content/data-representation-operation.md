<h1><center>第二章 数据的表示和运算</center></h1>

## 1. 本章定位

数据的表示和运算是计算机组成原理中最基础、也最容易出计算题的章节。

本章主要回答三个问题：

1. 计算机如何表示数值、字符和校验信息。
2. 计算机如何完成定点数的加、减、乘、除运算。
3. 计算机如何表示和运算浮点数。

408 中，本章常以选择题和综合应用题形式出现，重点考查数制转换、机器数表示范围、补码运算、溢出判断、加法器、浮点数表示和浮点数运算流程。

## 2. 本章内容

### 第一节 数制与编码

主要内容：

- 进位计数制
- 数制转换
- 真值与机器数
- 无符号数与带符号数
- 原码、反码、补码、移码
- BCD 码与字符编码
- 奇偶校验、海明码、循环冗余校验

### 第二节 运算方法和运算电路

主要内容：

- 定点数加减运算
- 补码加减法
- 溢出判断
- 移位运算
- 定点数乘法和除法
- 半加器、全加器
- 串行进位加法器与并行进位加法器
- ALU 的基本思想

### 第三节 浮点数的表示与运算

主要内容：

- 浮点数表示格式
- 规格化浮点数
- IEEE 754 标准
- 阶码、尾数与偏置值
- 浮点数加减运算
- 舍入、溢出与下溢
- 浮点数与定点数的比较

## 3. 学习主线

本章建议按下面顺序理解：

$$
\text{数制转换}
\longrightarrow
\text{机器数编码}
\longrightarrow
\text{补码运算}
\longrightarrow
\text{运算电路}
\longrightarrow
\text{浮点数表示}
\longrightarrow
\text{浮点数运算}
$$

::: tip 重点
本章不能只背公式。408 更常考“给一个具体数，让你表示、运算、判断溢出或分析结果”。复习时要多动手写二进制位串。
:::

## 4. 常见考点

| 考点 | 要求 |
| --- | --- |
| 数制转换 | 会在二、八、十、十六进制之间转换 |
| 机器数范围 | 会判断无符号数、原码、反码、补码、移码的范围 |
| 补码表示 | 会表示负数补码，理解补码只有一个零 |
| 符号扩展 | 会对补码进行正确扩展 |
| 补码加减法 | 会把减法转化为加法 |
| 溢出判断 | 会用符号位或进位判断溢出 |
| 加法器 | 理解半加器、全加器、进位产生与传递 |
| 浮点数格式 | 掌握符号位、阶码、尾数的含义 |
| IEEE 754 | 会分析单精度浮点数的位段和数值 |
| 浮点运算 | 掌握对阶、尾数运算、规格化、舍入流程 |

## 5. 易错点

- 真值和机器数不是同一个概念。
- 原码和反码都有正零、负零，补码只有一个零。
- $n$ 位补码整数的最小值是 $-2^{n-1}$，不是 $-(2^{n-1}-1)$。
- 补码符号扩展时补的是符号位。
- 溢出只可能发生在两个同号数相加或两个异号数相减时。
- 进位不等于溢出，最高位进位本身不能直接说明溢出。
- IEEE 754 的阶码通常采用偏置表示，不是普通补码。
- 浮点数运算时，小阶向大阶对齐。
