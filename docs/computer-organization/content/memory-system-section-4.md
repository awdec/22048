<h1><center>第四节 外部存储器</center></h1>

## 1. 外部存储器的作用

外部存储器简称外存，用于长期保存大量程序和数据。

外存特点：

- 容量大。
- 成本低。
- 断电后信息不丢失。
- 速度慢于主存。
- CPU 不能像访问主存一样直接访问外存中的数据。

程序和数据通常先从外存调入主存，CPU 再进行访问。

## 2. 常见外部存储器

| 类型 | 特点 |
| --- | --- |
| 磁盘 | 容量大，成本低，机械寻址，访问延迟较高 |
| 固态硬盘 | 基于 Flash，随机访问性能好，无机械部件 |
| 光盘 | 利用光学方式读写，常用于只读或归档 |
| 磁带 | 顺序访问，适合备份和归档 |

408 中最常考的是磁盘结构和磁盘访问时间。

## 3. 磁盘基本结构

磁盘由盘片、磁道、扇区、柱面和磁头等组成。

| 概念 | 含义 |
| --- | --- |
| 盘片 | 用于存储数据的圆形磁性介质 |
| 盘面 | 盘片的一个可存储表面 |
| 磁道 | 盘面上一圈同心圆 |
| 扇区 | 磁道被划分成的固定大小区域 |
| 柱面 | 多个盘面上半径相同的磁道集合 |
| 磁头 | 读写磁盘数据的部件 |

磁盘读写时，磁头先移动到指定磁道，再等待目标扇区旋转到磁头下方，最后传输数据。

## 4. 磁盘容量计算

磁盘容量常用公式：

$$
\text{磁盘容量}
=
\text{盘面数}
\times
\text{每面磁道数}
\times
\text{每道扇区数}
\times
\text{每扇区字节数}
$$

例如，某磁盘有 $4$ 个盘面，每个盘面 $1024$ 个磁道，每个磁道 $64$ 个扇区，每个扇区 $512\text{ B}$，则容量为：

$$
4 \times 1024 \times 64 \times 512\text{ B}
$$

计算时注意单位换算：

$$
1\text{ KB} = 2^{10}\text{ B}
$$

$$
1\text{ MB} = 2^{20}\text{ B}
$$

$$
1\text{ GB} = 2^{30}\text{ B}
$$

## 5. 磁盘访问时间

磁盘访问时间通常包括：

- 寻道时间
- 旋转延迟
- 传输时间

公式为：

$$
\text{访问时间}
=
\text{寻道时间}
+
\text{旋转延迟}
+
\text{传输时间}
$$

### 5.1 寻道时间

寻道时间是磁头移动到目标磁道所需时间。

寻道时间与磁头当前位置和目标磁道位置有关。

### 5.2 旋转延迟

旋转延迟是等待目标扇区旋转到磁头下方的时间。

若磁盘转速为 $R$ 转每秒，则旋转一周时间为：

$$
\frac{1}{R}
$$

平均旋转延迟通常取半圈时间：

$$
\frac{1}{2R}
$$

若转速用 rpm 表示，即每分钟转数，则每秒转数为：

$$
\frac{rpm}{60}
$$

### 5.3 传输时间

传输时间是数据从磁盘传出或写入磁盘所需时间。

若一个磁道有 $S$ 个扇区，磁盘转一圈时间为 $T$，读取 $n$ 个连续扇区，则传输时间为：

$$
\frac{n}{S} \times T
$$

## 6. 磁盘调度

磁盘调度属于操作系统中的常见内容，但在计组中理解磁盘访问过程也很有帮助。

常见调度思想：

- 先来先服务
- 最短寻道时间优先
- 扫描算法
- 循环扫描算法

它们的目标通常是减少平均寻道时间。

## 7. 固态硬盘

固态硬盘 SSD 通常基于 Flash 存储器。

特点：

- 无机械部件。
- 随机访问速度快。
- 抗震性好。
- 读写以页或块为单位。
- 擦除通常以块为单位。
- 写入寿命有限。

SSD 与磁盘对比：

| 项目 | 磁盘 | SSD |
| --- | --- | --- |
| 存储介质 | 磁性介质 | Flash |
| 是否有机械部件 | 有 | 无 |
| 随机访问 | 慢 | 快 |
| 顺序读写 | 较好 | 较好 |
| 擦写寿命 | 通常不是主要限制 | 有擦写次数限制 |

## 8. 外存与主存比较

| 项目 | 主存 | 外存 |
| --- | --- | --- |
| CPU 是否直接访问 | 可以 | 通常不可以 |
| 速度 | 快 | 慢 |
| 容量 | 较小 | 大 |
| 断电保存 | 通常不能 | 可以 |
| 作用 | 存放当前运行程序和数据 | 长期保存程序和数据 |

外存中的程序必须调入主存后，CPU 才能执行。

## 9. 408 常见考法

| 考法 | 示例 |
| --- | --- |
| 磁盘结构 | 判断磁道、扇区、柱面含义 |
| 容量计算 | 根据盘面、磁道、扇区、扇区大小计算容量 |
| 访问时间 | 计算寻道、旋转延迟和传输时间 |
| 转速换算 | rpm 与每秒转数互换 |
| SSD 特点 | 判断 SSD 与磁盘的差异 |

## 10. 易错点

- 平均旋转延迟通常是旋转半圈时间。
- rpm 是每分钟转数，计算时要除以 $60$。
- 磁盘访问时间不只包括数据传输时间。
- 柱面是不同盘面上相同半径磁道的集合。
- 外存容量大但 CPU 通常不能直接执行外存中的程序。
- SSD 没有机械寻道时间，但仍有读写和擦除开销。

## 11. 本节小结

外存重点掌握磁盘。磁盘题最常考容量和访问时间，做题时先分清盘面、磁道、扇区，再把寻道时间、旋转延迟和传输时间分别算清楚。
