<h1><center>第一章 计算机系统概述</center></h1>

## 1. 本章定位

计算机系统概述是计算机组成原理的入口章节，主要回答三个问题：

1. 计算机系统由哪些层次组成。
2. 硬件和软件如何协同工作。
3. 如何评价一台计算机的性能。

本章内容看似偏概念，但 408 中经常通过性能指标计算、层次结构判断、冯·诺依曼机特点辨析等方式考查。

## 2. 本章内容

### 第一节 计算机系统层次结构

主要内容：

- 计算机系统的组成
- 硬件系统与软件系统
- 计算机系统的层次结构
- 冯·诺依曼计算机
- 现代计算机的基本组成
- 计算机系统结构与计算机组成的区别

### 第二节 计算机的性能指标

主要内容：

- 机器字长
- 主频与时钟周期
- CPI
- 执行时间
- MIPS
- FLOPS
- 吞吐量与响应时间
- 存储容量与带宽

## 3. 学习主线

本章建议按下面顺序理解：

$$
\text{计算机系统}
\longrightarrow
\text{硬件与软件}
\longrightarrow
\text{系统层次结构}
\longrightarrow
\text{冯·诺依曼结构}
\longrightarrow
\text{性能指标}
$$

::: tip 重点
第一章不需要死记大量细节，核心是理解“层次”和“性能”。层次结构帮助理解后续指令系统、CPU、存储系统和 I/O 系统；性能指标则是后续计算题的基础。
:::

## 4. 常见考点

| 考点 | 要求 |
| --- | --- |
| 冯·诺依曼结构 | 掌握存储程序思想和五大组成部分 |
| 软硬件层次 | 理解高级语言、操作系统、指令系统、硬件之间的关系 |
| 指令系统接口 | 理解 ISA 是软硬件之间的重要接口 |
| 机器字长 | 会区分机器字长、指令字长、存储字长 |
| 主频与时钟周期 | 会进行主频和周期之间的换算 |
| CPI 与执行时间 | 会使用公式计算程序执行时间 |
| MIPS 与 FLOPS | 理解指标含义及局限性 |
| 吞吐量与响应时间 | 能区分系统整体处理能力和单个任务完成时间 |
