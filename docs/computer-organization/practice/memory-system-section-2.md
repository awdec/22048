<h1><center>第二节 主存储器</center></h1>

::: info 例 1.
某一 SRAM 芯片，容量为 $1024 × 8$ 位，该芯片的地址引脚和数据引脚总数是（）。<br>
A. 8<br>
B. 10<br>
C. 18<br>
D. 13

:::

::: details 解析
答案：C。

存储芯片容量写成 $M \times W$ 位时：

- 地址引脚数由存储单元个数 $M$ 决定。
- 数据引脚数由每个单元位数 $W$ 决定。

本题有 $1024=2^{10}$ 个存储单元，因此需要 $10$ 根地址引脚；每个单元 $8$ 位，因此需要 $8$ 根数据引脚。

总数为：

$$
10+8=18
$$

所以选 C。

:::

::: info 例 2.
下面有关 DRAM 和 SRAM 存储芯片的叙述：<br>
I. DRAM 芯片的集成度比 SRAM 芯片的高<br>
II. DRAM 芯片的成本比 SRAM 芯片的高<br>
III. DRAM 芯片的速度比 SRAM 芯片的快<br>
IV. DRAM 芯片工作时需要刷新，SRAM 芯片工作时不需要刷新<br>
通常情况下，错误的是（ ）。<br>
A. I 和 II<br>
B. II 和 III<br>
C. III 和 IV<br>
D. I 和 IV

:::

::: details 解析
答案：B。

SRAM 和 DRAM 的典型比较如下：

| 项目 | SRAM | DRAM |
| --- | --- | --- |
| 存储元件 | 触发器 | 电容 |
| 速度 | 快 | 较慢 |
| 集成度 | 低 | 高 |
| 成本 | 高 | 低 |
| 是否刷新 | 不需要 | 需要 |

因此 I 正确，IV 正确。

II 说 DRAM 成本比 SRAM 高，错误；III 说 DRAM 速度比 SRAM 快，错误。

所以错误的是 II 和 III，选 B。

:::

::: info 例 3.
下列关于随机存储器的说法中，正确的是（ ）。<br>
A. 半导体 RAM 中的信息可读可写，且断电后仍能保持记忆<br>
B. DRAM 是易失性 RAM，而 SRAM 中的存储信息是不易失的<br>
C. 半导体 RAM 是易失性 RAM，但只要电源不断电，所存信息是不丢失的<br>
D. 半导体 RAM 是非易失性 RAM

:::

::: details 解析
答案：C。

RAM 的特点是可随机读写，但半导体 RAM 通常是易失性存储器。

A 错在“断电后仍能保持记忆”。  
B 错在 SRAM 也是易失性的，只是不需要刷新。  
D 错在把半导体 RAM 说成非易失性。

只要电源不断，并且 DRAM 正常刷新，RAM 中的信息就能保持，所以选 C。

:::

::: info 例 4.
下面有关 ROM 和 RAM 的叙述中，错误的是（ ）。<br>
A. RAM 是可读可写存储器，ROM 是只读存储器<br>
B. ROM 和 RAM 都采用随机访问方式进行读/写<br>
C. 系统的主存由 RAM 和 ROM 组成<br>
D. 系统的主存都用 DRAM 芯片实现

:::

::: details 解析
答案：D。

主存通常由 RAM 和少量 ROM 组成：

- RAM 用于存放正在运行的程序和数据。
- ROM 常用于存放固件、启动程序等。

RAM 和 ROM 都可以按地址随机访问。这里的“随机访问”强调任意地址访问时间基本相同，不等于都能像 RAM 一样随时写入。

D 说系统主存都用 DRAM 芯片实现，忽略了 ROM 等组成部分，因此错误，选 D。

:::

::: info 例 5.
下列说法正确的是（ ）。<br>
A. EPROM 是可改写的，因此可以作为随机存储器<br>
B. EPROM 是可改写的，但不能作为随机存储器<br>
C. EPROM 是不可改写的，因此不能作为随机存储器<br>
D. EPROM 只能改写一次，因此不能作为随机存储器

:::

::: details 解析
答案：B。

EPROM 是可擦除可编程 ROM，可用紫外线擦除后重新写入。

但 EPROM 仍属于 ROM 类存储器，不是 RAM。它不能像随机读写存储器那样在正常工作过程中方便、快速地任意改写。

因此“可改写”不等于“可以作为随机存储器”，选 B。

:::

::: info 例 6.
下列（ ）是动态半导体存储器的特点。<br>
I. 在工作中存储器内容会产生变化<br>
II. 每隔一定时间，需要根据原存内容重新写入一遍<br>
III. 一次完整的刷新过程需要占用两个存取周期<br>
IV. 一次完整的刷新过程只需要占用一个存取周期<br>
A. I、III<br>
B. II、III<br>
C. II、IV<br>
D. 只有 III

:::

::: details 解析
答案：C。

动态半导体存储器指 DRAM。它用电容保存电荷，电荷会泄漏，因此必须定期刷新。

II 正确：每隔一定时间，要按原内容重新写入。  
IV 正确：刷新通常可看作一次“读出并再生”的存储周期。

I 不严谨，刷新不应改变逻辑内容；III 错在把一次刷新说成必须占用两个存取周期。

所以选 C。

:::

::: info 例 7.
下列关于 Flash 存储器特性与应用的说法中，正确的是（ ）。<br>
A. 读写速度一致，均接近 DRAM 的访问速率<br>
B. 属于易失性存储器，断电后数据丢失<br>
C. 写操作前需先擦除目标块，故写速慢于读速<br>
D. 仅能用作辅助存储器（如 U 盘）

:::

::: details 解析
答案：C。

Flash 是非易失性半导体存储器，断电后数据不丢失。

它的典型特点是：

- 读速度通常快于写速度。
- 写入前通常需要先按块擦除。
- 可用于 U 盘、SSD、嵌入式系统存储等，不只作为辅助存储器。

因此 C 正确，选 C。

:::

::: info 例 8.
DRAM 具有破坏性读出的特性，需要定时刷新，下列说法中不正确的是（ ）。<br>
A. 刷新是以行为单位的<br>
B. 刷新是为了给 DRAM 存储单元中的存储电容重新充电<br>
C. 刷新是通过对存储单元进行“读但不输出数据”，即“假读”的操作来实现的<br>
D. DRAM 内部设有专门的刷新电路，不会影响到 CPU 的正常访存

:::

::: details 解析
答案：D。

DRAM 刷新要抓住三点：

1. 刷新通常以行为单位进行。
2. 刷新本质是给存储电容重新充电。
3. 刷新可通过类似“读但不向 CPU 输出”的假读操作完成。

刷新期间会占用存储器内部资源，可能影响 CPU 正常访存，并不是完全无影响。

所以不正确的是 D。

:::

::: info 例 9.
下列关于 DRAM 和 SDRAM 的说法中，不正确的是（ ）。<br>
A. 传统 DRAM 芯片与 CPU 采用异步方式交换数据<br>
B. SDRAM 芯片与 CPU 采用同步方式交换数据<br>
C. DRAM 需要定期刷新，而 SDRAM 不需要定期刷新<br>
D. SDRAM 的行缓冲器通常用 SRAM 实现

:::

::: details 解析
答案：C。

SDRAM 的 S 表示 synchronous，即同步。它与系统时钟同步交换数据；传统 DRAM 通常采用异步方式。

但 SDRAM 仍然属于 DRAM，本质上仍用电容保存信息，因此仍需要定期刷新。

C 说 SDRAM 不需要定期刷新，错误，选 C。

:::

::: info 例 10.
每推出新一代 DRAM 芯片，地址引脚至少增加 1 根，则容量至少提高到原来的（ ）倍。<br>
A. 2<br>
B. 4<br>
C. 8<br>
D. 16

:::

::: details 解析
答案：B。

DRAM 通常采用行地址和列地址分时复用，同一组地址引脚先送行地址，再送列地址。

若地址引脚增加 $1$ 根，则行地址和列地址都可以多 $1$ 位，相当于总地址位数增加 $2$ 位。

容量至少变为：

$$
2^2=4
$$

倍，所以选 B。

:::

::: info 例 11.
某 DRAM 芯片容量为 $4\mathrm{M} \times 4$ 位，下列说法中错误的是（ ）。<br>
A. 芯片有 11 个地址引脚<br>
B. 芯片内部的行地址生成器（刷新计数器）为 11 位<br>
C. 芯片内部的行缓冲器容量为 2 Kb<br>
D. 芯片有 4 个数据引脚

:::

::: details 解析
答案：C。

$4\mathrm{M} \times 4$ 位表示有：

$$
4\mathrm{M}=2^{22}
$$

个地址单元，每个单元 $4$ 位。

DRAM 行、列地址复用，通常按近似方阵组织：

$$
2^{22}=2^{11}\times 2^{11}
$$

因此地址引脚为 $11$ 根，刷新计数器按行计数，为 $11$ 位；数据引脚为 $4$ 根。

行缓冲一次保存一整行数据。若按 $4$ 位输出宽度看，一行为：

$$
2^{11}\times 4=8192\text{ bit}=8\text{ Kb}
$$

不是 $2\text{ Kb}$。

所以错误的是 C。

:::

::: info 例 12.
已知单个存储体的存取周期为 110 ns，总线传输周期为 10 ns，采用低位交叉编址的多模块存储器时，存储体数应（ ）。<br>
A. 小于 11<br>
B. 等于 11<br>
C. 大于 11<br>
D. 大于或等于 11

:::

::: details 解析
答案：D。

低位交叉编址要让连续访问不断流，存储体数 $m$ 通常应满足：

$$
m \ge \frac{T}{\tau}
$$

其中 $T$ 是单个存储体的存取周期，$\tau$ 是总线传输周期。

本题：

$$
\frac{110\text{ ns}}{10\text{ ns}}=11
$$

因此存储体数应大于或等于 $11$，选 D。

:::

::: info 例 13.
一个四体并行低位交叉存储器，每个模块的容量是 64K × 32 位，存取周期为 200 ns，总线周期为 50 ns，在下述说法中，（ ）是正确的。<br>
A. 在 200 ns 内，存储器能向 CPU 提供 256 位二进制信息<br>
B. 在 200 ns 内，存储器能向 CPU 提供 128 位二进制信息<br>
C. 在 50 ns 内，每个模块能向 CPU 提供 32 位二进制信息<br>
D. 以上都不对

:::

::: details 解析
答案：B。

四体低位交叉存储器中，连续地址依次分布在 $4$ 个模块中。

本题每个模块一次提供 $32$ 位，存取周期为 $200\text{ ns}$，总线周期为 $50\text{ ns}$，恰好满足：

$$
200\text{ ns}=4\times 50\text{ ns}
$$

因此在一个 $200\text{ ns}$ 的周期内，可以依次从 $4$ 个模块得到 $4$ 个 $32$ 位字：

$$
4\times 32=128\text{ bit}
$$

所以选 B。

:::

::: info 例 14.
某机器采用四体低位交叉存储器，现分别执行下述操作：① 读取 6 个连续地址单元中存放的存储字，重复 80 次；② 读取 8 个连续地址单元中存放的存储字，重复 60 次。则①、②所花费的时间之比为（ ）。<br>
A. 1 : 1<br>
B. 2 : 1<br>
C. 4 : 3<br>
D. 3 : 4

:::

::: details 解析
答案：C。

四体低位交叉存储器可把连续地址分散到 $4$ 个存储体中。按“每轮最多访问 $4$ 个连续字”计：

- 读取 $6$ 个连续字，需要 $2$ 轮。
- 读取 $8$ 个连续字，也需要 $2$ 轮。

于是两种操作的单次读取轮数相同，时间只与重复次数成正比：

$$
2\times 80 : 2\times 60 = 4:3
$$

所以选 C。

:::

::: info 例 15.
假定用若干 $16\text{K} \times 8$ 的存储芯片组成一个 $64\text{K} \times 8$ 位的存储器，芯片各单元采用交叉编址方式，则地址 BFFFH 所在的芯片的最小地址为（ ）。<br>
A. 0000H<br>
B. 0001H<br>
C. 0002H<br>
D. 0003H

:::

::: details 解析
答案：D。

$64\text{K} \times 8$ 由 $16\text{K} \times 8$ 芯片组成，需要：

$$
\frac{64\text{K}}{16\text{K}}=4
$$

片。

采用交叉编址时，芯片号由低位地址决定：

$$
\text{芯片号}=A\bmod 4
$$

$\text{BFFFH}$ 的最低两位二进制为 $11$，因此所在芯片号为 $3$。

芯片 $3$ 中的最小主存地址是 $0003\text{H}$，所以选 D。

:::

::: info 例 16.
某存储器总线的宽度是 64 位，若用 8 个 16M × 8 位的 DRAM 芯片扩展构成 16M × 64 位的内存条，按字节编址，支持突发传送方式，某 double 型的变量 x 的主存地址为 2026 0000H，某 int 型的变量 y 的主存地址为 2026 1006H，则下列叙述中错误的是（ ）。<br>
A. 该内存条不可采用多模块交叉编址<br>
B. DRAM 芯片的行缓冲采用的是 SRAM<br>
C. 读取变量 x 只需要一个存取周期<br>
D. 读取变量 y 需要两个存取周期

:::

::: details 解析
答案：A。

存储器总线宽度为 $64$ 位，即一次可传送 $8$ 字节。

变量 $x$ 是 double 型，占 $8$ 字节，地址 $20260000\text{H}$ 的低 $3$ 位为 $000$，正好按 $8$ 字节边界对齐，因此读取 $x$ 只需一个存取周期。

变量 $y$ 是 int 型，占 $4$ 字节，地址 $20261006\text{H}$ 的块内偏移为 $6$。它占用偏移 $6,7$ 以及下一块的偏移 $0,1$，跨越两个 $8$ 字节传送块，因此需要两个存取周期。

DRAM 行缓冲常用 SRAM 结构实现。

这类由多个 $8$ 位芯片配合 $64$ 位总线构成的内存条，可以按字节交叉理解为多个模块并行提供不同字节。A 说“不可采用多模块交叉编址”，错误。

所以选 A。

:::

::: info 例 17.
【2012 统考真题】下列关于闪存的叙述中，错误的是（ ）。<br>
A. 信息可读可写，并且读、写速度一样快<br>
B. 存储元件由 MOS 管组成，是一种半导体存储器<br>
C. 掉电后信息不丢失，是一种非易失性存储器<br>
D. 采用随机访问方式，可替代计算机外部存储器

:::

::: details 解析
答案：A。

Flash 是半导体非易失性存储器，掉电后信息不丢失，可随机访问，也常用于替代外部存储器。

但 Flash 的读、写速度并不一样。写入通常慢于读取，并且写前往往需要先擦除。

因此错误的是 A。

:::

::: info 例 18.
【2014 统考真题】某容量为 256 MB 的存储器由若干 $4\text{M} \times 8$ 位的 DRAM 芯片构成，该 DRAM 芯片的地址引脚和数据引脚总数是（ ）。<br>
A. 19<br>
B. 22<br>
C. 30<br>
D. 36

:::

::: details 解析
答案：A。

$4\text{M} \times 8$ 表示有：

$$
4\text{M}=2^{22}
$$

个地址单元，每个单元 $8$ 位。

DRAM 采用行地址、列地址分时复用。通常将 $2^{22}$ 个单元组织为：

$$
2^{11}\times 2^{11}
$$

因此地址引脚数为 $11$，数据引脚数为 $8$。

地址引脚和数据引脚总数为：

::: info 例 19.
【2015 统考真题】某计算机使用四体交叉编址存储器，假定在存储器总线上出现的主存地址（十进制）序列为 8005, 8006, 8007, 8008, 8001, 8002, 8003, 8004, 8000，则可能发生访存冲突的地址对是（ ）。<br>
A. 8004 和 8008<br>
B. 8002 和 8007<br>
C. 8001 和 8008<br>
D. 8000 和 8004

:::

::: details 解析
答案：D。

四体交叉编址时，存储体号由地址的低 $2$ 位决定：

$$
\text{体号}=A\bmod 4
$$

各地址所在体号为：

| 地址 | 体号 |
| --- | --- |
| 8005 | 1 |
| 8006 | 2 |
| 8007 | 3 |
| 8008 | 0 |
| 8001 | 1 |
| 8002 | 2 |
| 8003 | 3 |
| 8004 | 0 |
| 8000 | 0 |

同一存储体连续或过近地被访问，可能发生访存冲突。

$8004$ 和 $8000$ 都在 $0$ 号体，并且在地址序列中相邻出现，因此可能发生冲突。

所以选 D。

:::

::: info 例 20.
【2017 统考真题】某计算机主存按字节编址，由 4 个 64M × 8 位的 DRAM 芯片采用交叉编址方式构成，并与宽度为 32 位的存储器总线相连，主存每次最多读/写 32 位数据。若 double 型变量 x 的主存地址为 804 001H，则读取 x 需要的存取周期数是（ ）。<br>
A. 1<br>
B. 2<br>
C. 3<br>
D. 4

:::

::: details 解析
答案：C。

主存总线宽度为 $32$ 位，一次最多读写 $4$ 字节。

double 型变量 $x$ 占 $8$ 字节。地址 $804001\text{H}$ 对 $4$ 取模，块内偏移为 $1$。

因此 $x$ 覆盖的字节范围跨越三个 $4$ 字节传送块：

| 传送块 | 覆盖情况 |
| --- | --- |
| 第 1 块 | 偏移 $1,2,3$ |
| 第 2 块 | 偏移 $0,1,2,3$ |
| 第 3 块 | 偏移 $0$ |

所以读取 $x$ 需要 $3$ 个存取周期，选 C。

:::

::: info 例 21.
【2018 统考真题】假定 DRAM 芯片中存储阵列的行数为 $r$、列数为 $c$，对于一个 $2\text{K} \times 1$ 位的 DRAM 芯片，为保证其地址引脚数最少，并尽量减少刷新开销，则 $r$、$c$ 的取值分别是（ ）。<br>
A. 2048, 1<br>
B. 64, 32<br>
C. 32, 64<br>
D. 1, 2048

:::

::: details 解析
答案：C。

$2\text{K} \times 1$ 位芯片共有：

$$
2\text{K}=2^{11}
$$

个存储位，因此：

$$
r\times c=2^{11}
$$

DRAM 行、列地址复用，地址引脚数由 $\max(\log_2 r,\log_2 c)$ 决定。要使地址引脚最少，应让行数和列数尽量接近。

候选中：

$$
64\times 32
\quad\text{和}\quad
32\times 64
$$

都只需要 $6$ 根地址引脚。

刷新按行进行，行数越少，刷新开销越小，因此选行数更少的：

$$
r=32,\quad c=64
$$

所以选 C。

:::

::: info 例 22.
【2022 统考真题】某内存条包含 8 个 8192 × 8192 × 8 位的 DRAM 芯片，按字节编址，支持突发（burst）传送方式，对应存储器总线宽度为 64 位，每个 DRAM 芯片内有一个行缓冲区（row buffer）。下列关于该内存条的叙述中，不正确的是（ ）。<br>
A. 内存条的容量为 512 MB<br>
B. 采用多模块交叉编址方式<br>
C. 芯片的地址引脚为 26 位<br>
D. 芯片内行缓冲有 8192 × 8 位

:::

::: details 解析
答案：C。

单个芯片容量为：

$$
8192\times 8192\times 8\text{ bit}
=2^{13}\times 2^{13}\times 8\text{ bit}
=2^{26}\text{ B}
=64\text{ MB}
$$

$8$ 个芯片总容量为：

$$
8\times 64\text{ MB}=512\text{ MB}
$$

所以 A 正确。

每个芯片数据宽度为 $8$ 位，$8$ 个芯片并行正好组成 $64$ 位存储器总线，可按多模块交叉方式提供连续字节，所以 B 正确。

芯片内部有 $8192=2^{13}$ 行、$8192=2^{13}$ 列。DRAM 行、列地址分时复用，因此芯片地址引脚应为 $13$ 位，而不是 $26$ 位，所以 C 错误。

一行包含 $8192$ 个列位置，每列 $8$ 位，因此行缓冲大小为：

$$
8192\times 8\text{ bit}
$$

D 正确。

本题选不正确的，故选 C。

:::

::: info 例 23.
下列关于单体多字存储器的说法中，不正确的是（ ）。<br>
A. 单体多字存储器主要解决主存容量太小的问题<br>
B. 单体多字存储器中，每个存储单元存储多个字<br>
C. 指令与数据的连续存放有利于单体多字存储器提高主存的读/写速度<br>
D. 过多的转移指令会严重影响单体多字存储器的工作效率

:::

::: details 解析
答案：A。

单体多字存储器的核心思想是：

$$
\text{一次访存读出多个连续字}
$$

它主要利用程序访问的空间局部性，提高连续取指或连续读写数据时的主存带宽。

B 正确：一个存储单元中可包含多个字，一次读出多个字。  
C 正确：指令和数据连续存放时，更容易一次取到后续要用的内容。  
D 正确：转移指令会破坏顺序访问，使一次取出的多个连续字可能用不上，从而降低效率。

A 错在“主要解决主存容量太小”。单体多字存储器重点解决的是主存访问速度或带宽问题，不是容量问题。

所以选 A。

:::

::: info 例 24.
【2011 统考真题】下列各类存储器中，不采用随机存取方式的是（ ）。<br>
A. EPROM<br>
B. CD-ROM<br>
C. DRAM<br>
D. SRAM

:::

::: details 解析
答案：B。

随机存取方式强调：

$$
\text{任意存储单元的访问时间基本相同}
$$

EPROM、DRAM、SRAM 都属于半导体存储器，可以按地址随机访问。

CD-ROM 属于光存储器。访问数据时需要机械定位、寻道或旋转等待，不同位置的访问时间不同，因此不采用随机存取方式。

所以选 B。

:::

## 题目总结

本组例题主要围绕主存芯片特性、DRAM 刷新、地址引脚、行缓冲、多模块交叉编址和总线传输展开。

| 例题 | 考查重点 | 核心结论 |
| --- | --- | --- |
| 例 1 | SRAM 芯片引脚数 | 地址引脚看单元数，数据引脚看位宽 |
| 例 2 | SRAM 与 DRAM 对比 | DRAM 集成度高、成本低、速度较慢、需要刷新 |
| 例 3 | RAM 易失性 | SRAM 和 DRAM 都是易失性存储器 |
| 例 4 | ROM 与 RAM 辨析 | 主存可由 RAM 和 ROM 共同组成 |
| 例 5 | EPROM 属性 | EPROM 可擦写，但仍属于 ROM 类存储器 |
| 例 6 | DRAM 刷新特点 | DRAM 需要按原内容周期性再生 |
| 例 7 | Flash 特性 | Flash 非易失，写前常需擦除，写慢于读 |
| 例 8 | DRAM 破坏性读出 | 刷新可能占用存储器资源，影响正常访存 |
| 例 9 | SDRAM 辨析 | SDRAM 仍是 DRAM，仍需刷新 |
| 例 10 | DRAM 地址复用 | 地址引脚增加 $1$ 根，容量通常至少变为 $4$ 倍 |
| 例 11 | DRAM 行缓冲 | 行缓冲容量等于一整行的数据量 |

| 例 13 | 四体低位交叉带宽 | 一个存取周期内可流水得到多个模块的数据 |
| 例 14 | 连续访存时间比 | 低位交叉按轮次和重复次数综合计算 |
| 例 15 | 交叉编址芯片号 | 芯片号由低位地址决定，常用 $A\bmod m$ |
| 例 16 | 总线宽度与对齐 | 跨越总线传送块时需要多个存取周期 |
| 例 17 | Flash 统考辨析 | Flash 可读写，但读写速度不一样 |
| 例 18 | DRAM 引脚数 | DRAM 行、列地址复用，地址引脚数取一半地址位 |
| 例 19 | 四体交叉冲突 | 同一存储体过近访问可能发生访存冲突 |
| 例 20 | 非对齐 double 读取 | 按总线宽度划分传送块，判断跨几个块 |

这些题共同强调：

$$
\text{先判断存储器类型，再判断地址组织，最后计算传输次数}
$$

做题时尤其要分清：

- SRAM、DRAM、ROM、Flash 的基本属性。
- 芯片容量中的“单元数”和“每单元位数”。
- DRAM 的行、列地址是否复用。
- 多体交叉编址中模块号由高位还是低位决定。
- 数据是否跨越总线一次传送的边界。
- 存储器采用随机存取、顺序存取还是直接存取。

## 答题技巧总结

### 1. 存储器类型辨析题

先按下面表格快速定位：

| 类型 | 是否易失 | 典型特点 | 常见用途 |
| --- | --- | --- | --- |
| SRAM | 易失 | 快、不刷新、成本高 | Cache |
| DRAM | 易失 | 容量大、成本低、需刷新 | 主存 |
| ROM | 非易失 | 读为主，写入受限 | 固件 |
| Flash | 非易失 | 可电擦写，写慢于读 | SSD、U 盘 |

易错点是：

$$
\text{SRAM 不需要刷新}\ne\text{SRAM 非易失}
$$

只要断电，SRAM 和 DRAM 中的信息都会丢失。

### 2. 芯片容量和引脚数题

芯片规格一般写成：

$$
M\times W
$$

其中：

- $M$ 决定地址空间。
- $W$ 决定一次输出或输入的数据位数。

若是普通 RAM：

$$
\text{地址引脚数}=\log_2 M
$$

若是 DRAM，常见行、列地址复用：

$$
\text{地址引脚数}
=
\max(\log_2 r,\log_2 c)
$$

数据引脚数通常直接等于位宽 $W$。

### 3. DRAM 刷新和行缓冲题

DRAM 做题抓三句话：

1. DRAM 用电容保存信息，所以需要刷新。
2. 刷新通常以行为单位。
3. 行缓冲保存一整行数据。

若芯片组织为：

$$
r\times c\times W
$$

则一行数据量通常为：

$$
c\times W
$$

减少刷新开销时，要优先减少行数 $r$。

### 4. 多模块交叉编址题

低位交叉编址的核心公式是：

$$
\text{模块号}=A\bmod m
$$

其中 $m$ 为模块数。

连续访存时，若要流水不断流，通常要求：

$$
m\ge\frac{T}{\tau}
$$

其中：

- $T$ 是单个模块的存取周期。
- $\tau$ 是总线传输周期。

判断冲突时，先把地址逐个对 $m$ 取模，看是否短时间内访问同一个模块。

### 5. 总线宽度和跨块访问题

按字节编址时，若总线宽度为 $W$ 位，则一次最多传送：

$$
\frac{W}{8}\text{ B}
$$

判断某个变量需要几个存取周期，按下面步骤：

1. 求一次传送的字节数。
2. 看变量起始地址相对于该传送块的偏移。
3. 看变量覆盖了几个传送块。
4. 覆盖几个传送块，通常就需要几个存取周期。

例如 $32$ 位总线一次传送 $4$ 字节，若一个 $8$ 字节变量从块内偏移 $1$ 开始，就会覆盖 $3$ 个传送块。

### 6. 单体多字和存取方式辨析题

单体多字存储器不是为了扩大容量，而是为了提高连续访问效率：

$$
\text{一次访存}
\longrightarrow
\text{取出多个连续字}
$$

它依赖空间局部性，所以顺序取指、顺序访问数组时效果较好；若程序中转移指令过多，连续取出的字可能用不上，效率会下降。

存取方式辨析时，可以按下面表格记：

| 存取方式 | 特点 | 典型例子 |
| --- | --- | --- |
| 随机存取 | 任意地址访问时间基本相同 | SRAM、DRAM、EPROM |
| 顺序存取 | 必须按顺序访问 | 磁带 |
| 直接存取 | 先定位到区域，再顺序访问 | 磁盘、光盘 |
| 相联存取 | 按内容访问 | Cache 标记、TLB |

看到 CD-ROM、磁盘这类有机械定位或旋转等待的存储器，不能按半导体 RAM 的随机存取方式处理。
