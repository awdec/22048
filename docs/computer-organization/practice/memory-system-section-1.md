<h1><center>第一节 存储器概述</center></h1>

::: info 例 1.
磁盘属于（ ）类型的存储器。<br>
A. 随机存储器（RAM）<br>
B. 只读存储器（ROM）<br>
C. 顺序存取存储器（SAM）<br>
D. 直接存取存储器（DAM）
:::

::: details 解析
答案：D。

按存取方式分类：

| 类型 | 访问特点 | 典型例子 |
| --- | --- | --- |
| 随机存取 | 任意单元访问时间基本相同 | RAM |
| 顺序存取 | 必须按顺序访问 | 磁带 |
| 直接存取 | 先定位到附近区域，再顺序访问 | 磁盘、光盘 |
| 相联存取 | 按内容访问 | Cache 标记、TLB |

磁盘访问数据时，通常要先定位到磁道、扇区附近，再顺序读写目标数据，因此属于直接存取存储器。

所以选 D。

:::

::: info 例 2.
存储器的存取周期是指（ ）。<br>
A. 存储器的读出时间<br>
B. 存储器的写入时间<br>
C. 存储器进行连续读 / 写操作所允许的最短时间间隔<br>
D. 存储器进行一次读 / 写操作所需的平均时间
:::

::: details 解析
答案：C。

存取时间和存取周期要分开：

- 存取时间：完成一次读或写操作所需的时间。
- 存取周期：连续两次独立访问存储器之间允许的最短时间间隔。

由于一次访问结束后，存储器可能还需要恢复、刷新或准备下一次访问，所以通常有：

$$
\text{存取周期}\ge \text{存取时间}
$$

因此本题选 C。

:::

::: info 例 3.
相联存储器是一种特殊的存储器，其主要特点是（ ）。<br>
A. 通过地址总线指定存储单元进行读 / 写<br>
B. 按照 “后进先出” 原则访问数据<br>
C. 根据存储内容进行并行匹配查找<br>
D. 仅用于实现高速缓存中的直接映射结构
:::

::: details 解析
答案：C。

相联存储器也称按内容访问存储器，其核心不是“给地址找内容”，而是：

$$
\text{给出关键内容}
\longrightarrow
\text{并行比较}
\longrightarrow
\text{找到匹配项}
$$

A 描述的是普通按地址访问的存储器。  
B 描述的是栈。  
D 错在“仅用于直接映射”，相联思想常见于 Cache 标记匹配、TLB 等场景。

所以选 C。

:::

::: info 例 4.
下列关于多级存储系统的说法中，正确的有（ ）。<br>
I. 多级存储系统是为了降低存储成本<br>
II. 虚拟存储器中主存和辅存之间的数据调动对任何程序员是透明的<br>
III. CPU 只能与 Cache 直接交换信息，CPU 与主存交换信息也需要经过 Cache<br>
A. 仅 I<br>
B. 仅 I 和 II<br>
C. I、II 和 III<br>
D. 仅 II
:::

::: details 解析
答案：A。

逐项判断：

I 正确。多级存储系统通过“少量高速存储器 + 大量低速低价存储器”组合，提高性能价格比，可以降低平均位成本。更完整地说，它是为了解决速度、容量和成本之间的矛盾。

II 错误。虚拟存储器中主存和辅存之间的数据调动通常对应用程序员透明，但不是对任何程序员都透明。操作系统和相关系统程序需要参与管理。

III 错误。Cache 是 CPU 和主存之间的高速缓冲，但不能说 CPU 只能与 Cache 直接交换信息。发生 Cache 未命中时，仍需要从主存取得数据；Cache 和主存之间的数据调动由硬件自动完成。

因此只有 I 正确，选 A。

:::

## 题目总结

本组例题主要围绕存储器概述中的分类、性能指标、相联访问和多级存储系统展开。

| 例题 | 考查重点 | 核心结论 |
| --- | --- | --- |
| 例 1 | 存取方式分类 | 磁盘属于直接存取存储器 |
| 例 2 | 存取周期 | 存取周期是连续两次独立访存允许的最短时间间隔 |
| 例 3 | 相联存储器 | 相联存储器按内容进行并行匹配查找 |
| 例 4 | 多级存储系统 | 多级存储系统用于协调速度、容量和成本的矛盾 |

这些题共同强调：

$$
\text{先分类，再抓定义；先看访问方式，再看系统层次}
$$

本节容易混淆的概念有：

- 随机存取、顺序存取、直接存取和相联存取。
- 存取时间和存取周期。
- 按地址访问和按内容访问。
- Cache 层次和虚拟存储器层次中的“透明性”。

## 答题技巧总结

### 1. 存取方式题

看到“属于哪类存储器”，先按访问方式分类：

| 存取方式 | 特点 | 典型例子 |
| --- | --- | --- |
| 随机存取 | 任意地址访问时间基本相同 | RAM |
| 顺序存取 | 必须按顺序访问 | 磁带 |
| 直接存取 | 先定位到附近区域，再顺序访问 | 磁盘、光盘 |
| 相联存取 | 按内容匹配访问 | Cache 标记、TLB |

磁盘不是随机存取，也不是顺序存取，而是直接存取。

### 2. 性能指标题

存取时间和存取周期要分清：

$$
\text{存取周期}\ge \text{存取时间}
$$

存取时间强调“一次操作用多久”；存取周期强调“下一次独立操作最早什么时候能开始”。

### 3. 相联存储器题

相联存储器的关键词是：

$$
\text{按内容访问}
\quad
\text{并行比较}
\quad
\text{匹配查找}
$$

只要题目说“根据内容查找”“并行匹配”，通常就指向相联存储器。

### 4. 多级存储系统题

多级存储系统的目的不是单纯追求某一个指标，而是综合平衡：

$$
\text{速度}
\quad
\text{容量}
\quad
\text{成本}
$$

判断透明性时要注意对象：

- Cache 对程序员通常透明。
- 虚拟存储器对应用程序员通常透明。
- 操作系统和系统程序并不一定透明。

看到“任何程序员”“只能”“必须”等绝对说法，要特别谨慎。
