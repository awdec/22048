<h1><center>第一节 计算机系统层次结构</center></h1>

::: info 题 1.
冯·诺依曼机的基本工作方式是（）。<br>
A. 控制流驱动方式<br>
B. 多指令多数据流方式<br>
C. 微程序控制方式<br>
D. 数据流驱动方式
:::

::: details 解析
答案：A。

冯·诺依曼机的核心思想是“存储程序”，程序和数据都存放在存储器中，CPU 按程序规定的指令序列取指、译码并执行。

这种工作方式本质上是由指令序列控制计算过程，即控制流驱动方式。

- B：多指令多数据流方式是并行计算机的一种分类，不是冯·诺依曼机的基本工作方式。
- C：微程序控制方式是控制器的一种实现方法，不是整机基本工作方式。
- D：数据流驱动方式强调数据就绪后触发运算，不是传统冯·诺依曼机的基本特点。

:::

::: info 题 2.
冯·诺依曼机工作方式的基本特点是（）。<br>
A. 程序一边被输入计算机一边被执行<br>
B. 程序直接从磁盘读到 CPU 执行<br>
C. 按地址访问指令并自动按序执行程序<br>
D. 程序自动执行而数据手工输入
:::

::: details 解析
答案：C。

冯·诺依曼机采用存储程序方式，程序预先以二进制形式存放在存储器中。执行时，CPU 按地址从存储器中取出指令，并按指令序列自动执行。

因此其基本特点可以概括为：

$$
\text{按地址取指}
\longrightarrow
\text{自动按序执行}
$$

遇到转移指令时，程序执行顺序可以改变，但仍然是通过改变下一条指令地址来实现的。

- A：程序不是“一边输入一边执行”的简单输入流。
- B：磁盘中的程序不能被 CPU 直接执行，通常要装入主存。
- D：数据也可以由程序和输入设备自动提供，不是“手工输入”作为基本特点。

:::

::: info 题 3.
计算机系统采用层次化结构，从最上层的应用程序到最底层的硬件，其典型层次自上而下依次为（ ）。<br>
A. 高级语言虚拟机→操作系统虚拟机→汇编语言虚拟机→机器语言机器<br>
B. 高级语言虚拟机→汇编语言虚拟机→机器语言机器→操作系统虚拟机<br>
C. 高级语言虚拟机→汇编语言虚拟机→操作系统虚拟机→机器语言机器<br>
D. 操作系统虚拟机→高级语言虚拟机→汇编语言虚拟机→机器语言机器

:::

::: details 解析
答案：C。

计算机系统可以看成由多级虚拟机组成。典型层次从上到下可理解为：

$$
\text{高级语言虚拟机}
\longrightarrow
\text{汇编语言虚拟机}
\longrightarrow
\text{操作系统虚拟机}
\longrightarrow
\text{机器语言机器}
\longrightarrow
\text{硬件}
$$

含义是：

- 高级语言程序先经编译、解释等方式向下转换。
- 汇编语言再由汇编程序转换为机器语言。
- 操作系统向上提供系统调用等服务，向下管理硬件资源。
- CPU 最终执行的是机器语言指令。

所以正确顺序为 C。

:::

::: info 题 4.
下列关于计算机系统层次结构的说法中，正确的是（ ）。<br>
A. 高级语言程序经编译生成汇编语言后，可直接在机器上执行<br>
B. ISA 仅定义指令功能，不涉及硬件实现细节<br>
C. 同一 ISA 可由不同微体系结构实现，软件无须修改即可兼容<br>
D. 高级语言中的每条语句与 ISA 的机器指令一一对应
:::

::: details 解析
答案：C。

同一 ISA 可以由不同的微体系结构实现。例如，同一种指令集可以用不同的数据通路、流水线结构、Cache 结构来实现。只要对程序员可见的机器属性保持一致，原有机器级程序通常可以兼容运行。

逐项分析：

- A 错误。汇编语言源程序不能被硬件直接执行，必须先由汇编程序转换成机器语言。
- B 表述不严谨。ISA 不只定义指令功能，还包括指令格式、寄存器、数据类型、寻址方式等程序员可见属性；但它通常不规定具体硬件实现细节。
- C 正确。同一 ISA 可以有不同微体系结构实现。
- D 错误。高级语言语句和机器指令通常不是一一对应关系，一条高级语言语句可能对应多条机器指令。

:::

::: info 题 5.
只有当程序执行时才将源程序翻译成机器语言，并且一次只能翻译一行语句，边翻译边执行的是（ ）程序，把汇编语言源程序转换为机器语言程序的过程是（ ）。<br>
I. 编译 II. 目标 III. 汇编 IV. 解释<br>
A. I、II<br>
B. IV、II<br>
C. IV、I<br>
D. IV、III
:::

::: details 解析
答案：D。

“执行时才翻译、一次翻译一行、边翻译边执行”描述的是解释程序。

因此第一空为：

$$
\text{解释程序}
$$

对应 IV。

把汇编语言源程序转换为机器语言程序的过程称为汇编，由汇编程序完成。

因此第二空为：

$$
\text{汇编}
$$

对应 III。

所以答案为：

$$
\text{IV、III}
$$

即 D。

注意区分：

| 名称 | 作用 |
| --- | --- |
| 编译 | 将高级语言程序翻译成目标程序 |
| 解释 | 边翻译边执行高级语言语句 |
| 汇编 | 将汇编语言程序翻译成机器语言程序 |
| 链接 | 将目标模块和库函数连接成可执行程序 |

:::

::: info 题 6.
【2015 统考真题】计算机硬件能够直接执行的是（ ）。<br>
I. 机器语言程序 II. 汇编语言程序 III. 硬件描述语言程序<br>
A. 仅 I<br>
B. 仅 I、II<br>
C. 仅 I、III<br>
D. I、II、III
:::

::: details 解析
答案：A。

计算机硬件能够直接识别并执行的是机器语言程序，也就是由二进制机器指令组成的程序。

逐项分析：

- I 正确。机器语言程序可以被 CPU 直接执行。
- II 错误。汇编语言程序只是机器语言的符号化表示，需要先经汇编程序转换为机器语言。
- III 错误。硬件描述语言用于描述和设计硬件电路，不是 CPU 直接执行的程序。

所以只有 I 正确，答案为 A。

:::

::: info 题 7.
【2016 统考真题】将高级语言源程序转换为机器级目标代码文件的程序是（ ）。<br>
A. 汇编语言程序<br>
B. 链接程序<br>
C. 编译程序<br>
D. 解释程序
:::

::: details 解析
答案：C。

将高级语言源程序转换为机器级目标代码文件的程序是编译程序。

逐项分析：

- A 错误。汇编语言程序本身是一种低级语言程序，不是完成高级语言翻译的工具。
- B 错误。链接程序负责把多个目标模块和库函数连接成可执行文件。
- C 正确。编译程序负责把高级语言源程序翻译为目标代码，目标代码可以是汇编代码或机器级目标代码。
- D 错误。解释程序边翻译边执行，一般不生成独立的机器级目标代码文件。

因此本题选 C。

:::

## 题目总结

本节题目主要围绕“计算机系统层次结构”中的概念辨析展开。

| 题号 | 考点 | 关键结论 |
| --- | --- | --- |
| 题 1 | 冯·诺依曼机工作方式 | 存储程序、控制流驱动 |
| 题 2 | 存储程序思想 | 按地址取指，自动按序执行 |
| 题 3 | 计算机系统层次结构 | 高级语言、汇编语言、操作系统、机器语言之间的层次关系 |
| 题 4 | ISA 与微体系结构 | 同一 ISA 可以由不同微体系结构实现 |
| 题 5 | 翻译程序 | 解释、编译、汇编、链接的作用区分 |
| 题 6 | 硬件直接执行对象 | CPU 只能直接执行机器语言程序 |
| 题 7 | 编译程序作用 | 高级语言源程序转换为机器级目标代码 |

本节最容易混淆的是三组概念：

| 易混概念 | 区分方法 |
| --- | --- |
| 汇编语言和机器语言 | 汇编语言是符号化表示，机器语言是二进制机器指令 |
| 编译和解释 | 编译先整体翻译，解释边翻译边执行 |
| ISA 和微体系结构 | ISA 是程序员可见接口，微体系结构是内部实现方式 |

## 答题技巧总结

第一章“计算机系统概述”题目通常不难，但特别喜欢考概念边界。做题时要抓住下面几条主线。

### 1. 看到“直接执行”，优先选机器语言

CPU 最终执行的是机器指令。

因此：

- 高级语言不能被 CPU 直接执行。
- 汇编语言不能被 CPU 直接执行。
- 机器语言程序可以被 CPU 直接执行。

记忆方式：

$$
\text{高级语言}
\longrightarrow
\text{汇编语言}
\longrightarrow
\text{机器语言}
\longrightarrow
\text{CPU 执行}
$$

### 2. 看到“边翻译边执行”，选解释程序

常见翻译程序要这样区分：

| 关键词 | 对应程序 |
| --- | --- |
| 高级语言翻译成目标代码 | 编译程序 |
| 边翻译边执行 | 解释程序 |
| 汇编语言翻译成机器语言 | 汇编程序 |
| 多个目标模块合成可执行文件 | 链接程序 |
| 可执行程序送入主存运行 | 装入程序 |

### 3. 看到“软件和硬件接口”，想到指令系统

指令系统也称 ISA，是软件和硬件之间的重要接口。

ISA 主要规定程序员可见的机器属性，例如：

- 指令格式
- 指令功能
- 寄存器
- 数据类型
- 寻址方式

但 ISA 不等于具体硬件实现。数据通路、流水线、Cache 结构等属于计算机组成或微体系结构层面。

### 4. 判断层次结构时，从“人”到“硬件”逐级下沉

常见顺序可以理解为：

$$
\text{用户}
\longrightarrow
\text{应用程序}
\longrightarrow
\text{操作系统}
\longrightarrow
\text{机器语言}
\longrightarrow
\text{微体系结构}
\longrightarrow
\text{硬件电路}
$$

若题目使用“虚拟机”表述，则常见顺序为：

$$
\text{高级语言虚拟机}
\longrightarrow
\text{汇编语言虚拟机}
\longrightarrow
\text{操作系统虚拟机}
\longrightarrow
\text{机器语言机器}
$$

### 5. 冯·诺依曼机抓住“存储程序”

冯·诺依曼机相关题目优先检查是否体现：

- 程序和数据都存放在存储器中。
- 程序由指令组成。
- CPU 按地址取指令。
- 指令控制计算机自动执行。

一句话概括：

$$
\text{存储程序}
=
\text{程序像数据一样存入存储器，由 CPU 自动取指执行}
$$

### 6. 概念判断题先排除“绝对化表述”

遇到下面说法要特别警惕：

- “高级语言可直接执行”
- “汇编语言可直接执行”
- “每条高级语言语句对应一条机器指令”
- “ISA 只定义指令功能”
- “主频越高计算机一定越快”

这类说法往往过于绝对，通常需要结合编译器、指令系统、CPI、指令条数等条件判断。
